## 一、货币性资产 vs 非货币性资产（判断基础）

|     类型     |     定义      |      大白话       |                常见举例                |
| :--------: | :---------: | :------------: | :--------------------------------: |
| **货币性资产**  | 金额固定或可确定的资产 | 未来收回的**钱是确定的** |        库存现金、银行存款、应收账款、应收票据         |
| **非货币性资产** |  金额不固定的资产   | 未来值多少钱**不确定**  | 存货、固定资产、无形资产、长期股权投资、投资性房地产、交易性金融资产 |

> ⚠️ **预付账款**属于**非货币性资产**（收回的是货物/服务，不是固定金额的货币）

---

## 二、非货币性资产交换的认定 ⭐

### 判定公式

```
收到的补价 ÷ 换出资产公允价值 ≤ 25%

或

支付的补价 ÷（换出资产公允价值 + 支付的补价）≤ 25%
```

|    比例     |      结论       |
| :-------: | :-----------: |
| **≤ 25%** | ✅ 属于非货币性资产交换  |
| **> 25%** | ❌ 不属于，按正常买卖处理 |

### ⚠️ 易错陷阱：补价比例计算必须**不含税差**

> 考纲特别标注的重点陷阱！

```
不含税的补价 = 实际支付的含税补价 - 增值税进销差额
```

**举例说明**：

假设换出资产公允100万（增值税13%），换入资产公允120万（增值税13%）：

```
不含税补价 = 120 - 100 = 20万

❌ 错误算法：22.6 ÷ 100 = 22.6%（用含税价）
✅ 正确算法：20 ÷ 100 = 20%（用不含税价）
```

如果换出资产和换入资产税率不同，用含税价算可能**误判结果**！

## 三、商业实质的判断

> 考试中通常题目会明示"具有商业实质"或"不具有"。

**具有商业实质**的判断标准（满足其一即可）：
1. 换入资产的未来现金流量在**风险、金额、时间**方面与换出资产**显著不同**
2. 换入资产与换出资产的**预计未来现金流量现值**不同，且差额重大

## 四、两种计价模式 ⭐⭐⭐

|           条件           |   计价基础   |     确认损益？      |
| :--------------------: | :------: | :------------: |
| **有商业实质 + 公允价值能可靠计量**  | **公允价值** | ✅ 确认**资产处置损益** |
| **无商业实质 或 公允价值不能可靠计量** | **账面价值** |    ❌ 不确认损益     |

## 五、会计处理（核心分录）

### 5.1 有商业实质 + 公允价值能可靠计量

#### 入账价值公式

```
换入资产入账价值 = 换出资产公允价值 + 支付的补价（- 收到的补价）+ 相关税费（不含增值税）
```

#### 换出资产视同销售 → 确认损益

```
损益 = 换出资产公允价值 - 换出资产账面价值
```

|     换出资产类型     |    对应损益科目     |
| :------------: | :-----------: |
|       存货       | 主营业务收入/主营业务成本 |
|   固定资产/无形资产    |  **资产处置损益**   |
| 长期股权投资/交易性金融资产 |   **投资收益**    |

#### 分录模板

**支付补价方（甲公司）：**

```
① 换出资产清理/转出
借：固定资产清理                  （账面价值）
    累计折旧
    贷：固定资产                       （原值）

② 换入资产入账
借：换入资产（如原材料/无形资产等）   （入账价值）
    应交税费——应交增值税（进项税额）
    贷：固定资产清理                   （换出资产公允价值）
        银行存款                       （支付的补价）
        应交税费——应交增值税（销项税额）

③ 确认换出资产损益
借：固定资产清理                  （公允-账面）
    贷：资产处置损益
```

---

### 📌 例题精讲

> 甲公司用一台设备换入乙公司一批原材料。设备原价200万元，已提折旧50万元，账面价值150万元，公允价值**180万元**。甲公司另向乙公司支付**不含税补价20万元**。该项交换具有**商业实质**，设备增值税税率13%，原材料增值税税率13%。

**第一步：判断是否属于非货币性资产交换**

```
补价比例 = 20 ÷（180 + 20）= 20 ÷ 200 = 10% ≤ 25% ✅
→ 属于非货币性资产交换
```

**第二步：计算换入资产入账价值**

```
换入原材料入账价值 = 换出设备公允 180万 + 支付补价 20万 = 200万元
```

**第三步：甲公司账务处理**

```
① 转入清理
借：固定资产清理              150万
    累计折旧                   50万
    贷：固定资产                   200万

② 换入原材料
借：原材料                    200万
    应交税费——应交增值税（进项税额）  26万（200×13%）
    贷：固定资产清理                 180万（设备公允价值）
        银行存款                     20万（补价）
        应交税费——应交增值税（销项税额）23.4万（180×13%）

③ 确认损益
借：固定资产清理             30万（180-150）
    贷：资产处置损益              30万
```

---

### 5.2 无商业实质 / 公允价值不能可靠计量

#### 入账价值公式

```
换入资产入账价值 = 换出资产账面价值 + 支付的补价（- 收到的补价）+ 相关税费
```

#### 不确认损益

换出资产直接按**账面价值**转出，不产生资产处置损益。

## 六、两种情况的对比

|  对比项目  | 公允价值模式（有商业实质+公允可靠） | 账面价值模式（无商业实质/公允不可靠） |
| :----: | :----------------: | :-----------------: |
| 换入资产计价 |  换出**公允**价+补价+税费   |   换出**账面**价+补价+税费   |
| 是否确认损益 |  ✅ **确认**（公允-账面）   |      ❌ **不确认**      |
|  换出资产  |     视同销售，按公允转出     |       按账面价值转出       |

## 八、易错点汇总 ⚠️

| 序号 | 易错点 | 正确做法 |
|:----:|:------:|:---------|

## 答疑解惑
这个问题问得很好！核心原因在于**固定资产清理这个科目存在的必要性**，而长投不需要。

---

## 原因：固定资产处置涉及多项费用，需要一个"中转站"

### 固定资产清理是什么？

**"固定资产清理"是一个汇集科目**，相当于一个"临时账本"，把处置过程中所有相关收支都归集到这里，最后轧出差额转到损益。

### 固定资产处置可能涉及哪些东西？

| 项目 | 说明 |
|:----:|:------|
| ① 固定资产原值和累计折旧 | 转进来 |
| ② **清理费用** | 比如拆机器的人工费、运输费 |
| ③ **残料收入** | 卖废铁的钱 |
| ④ **保险赔偿** | 如果资产报废有保险赔款 |
| ⑤ **应交税费** | 处置产生的增值税等 |
| ⑥ **最后差额** | 转到资产处置损益 |

### 对比一下就明白了

**情形A：换出固定资产（需要中转站）**

```
① 先把原值和折旧转进来
借：固定资产清理
    累计折旧
    贷：固定资产

② 可能发生清理费用
借：固定资产清理   （拆机器的运费等）
    贷：银行存款

③ 可能收到残料收入
借：银行存款
    贷：固定资产清理

④ 最后把差额转出（如果是非货币性交换，这里换入资产入账）
    借：固定资产清理    ← 设备公允价 180万
        贷：固定资产清理
        银行存款  等等

⑤ 最后固定资产清理这个临时账本的余额，转到损益
借：固定资产清理    （30万）
    贷：资产处置损益    （30万）
```

**固定资产清理是个"过程归集账户"，把处置全程的收支都集中在一起。**

---

**情形B：换出长期股权投资（不需要中转站）**

```
借：固定资产          600万  ← 换入资产
    贷：长期股权投资       500万  ← 长投直接转出
        投资收益           100万
```

**为什么长投不需要中转？** 因为：

| 对比项 | 固定资产 | 长期股权投资 |
|:------:|:--------:|:------------:|
| 处置过程 | 可能涉及拆除、运输、残料回收等**多步骤** | 签个合同、过户，**一步到位** |
| 额外费用 | 经常发生（运费、拆装费） | 极少有额外费用 |
| 历史原因 | 传统上一直用固定资产清理归集 | 金融资产处理，直接转 |

> **打个比方：**
> - 你家要卖一台旧冰箱 → 需要人搬、可能还要付搬运费 → 需要一个"中转站"记这些零碎开销
> - 你要卖你持有的阿里巴巴股票 → 手机上点一下卖出就行了 → 不需要中转站

---

## 但如果是"卖"而非"交换"呢？

**固定资产出售（收钱）：**

```
借：固定资产清理
    累计折旧
    贷：固定资产

借：银行存款              ← 收到钱
    贷：固定资产清理

借：固定资产清理          ← 差额到损益
    贷：资产处置损益
```

**长期股权投资出售（收钱）：**

```
借：银行存款              ← 直接收钱
    贷：长期股权投资           ← 直接转出
        投资收益               ← 直接算损益
```

长投从来不需要也不可能设一个"投资清理"，因为不存在"拆运费"这类步骤。

---

## 总结一句话

> **固定资产清理 = 处置过程中的"临时账本"**，因为固定资产处置可能涉及多项额外收支（运费、残料、保险等），需要一个科目来归集。而长期股权投资处置就是一步到位，直接贷记长投、差额进投资收益即可。

**考试中碰到换出固定资产：**

```
借：固定资产清理          ✅ 一定要先转清理
    累计折旧
    贷：固定资产

然后：
借：换入资产
    贷：固定资产清理
        银行存款等
        应交税费

最后：
借：固定资产清理
    贷：资产处置损益
```

**考试中碰到换出长期股权投资：**

```
借：换入资产
    贷：长期股权投资          ✅ 直接转，不用中转
        投资收益

同时：
借：其他综合收益（如有）
    贷：投资收益
```
