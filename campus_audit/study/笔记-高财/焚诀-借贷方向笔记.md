## 📐 核心心法：借贷方向统一公式

```
借：我得到的（新资产/费用）
贷：我付出的（旧资产/现金/负债/权益）
差额 → 损益类科目
```

**得到的 = 付出的价值**

|      要素       |      借方      |        贷方         |
| :-----------: | :----------: | :---------------: |
|    **我得到**    |    新资产入账     |         —         |
|    **我付出**    |      —       | 旧资产/现金/股票/债权的账面转出 |
| **公允价值≠账面价值** | 差额也可能在借方（损失） |   差额也可能在贷方（收益）    |

---

## ⚖️ 公允价值 vs 账面价值的选择

### 判定原则

> ✅ **独立公平交易 + 价值能可靠计量** → 用**公允价值**
> ❌ **关联内部交易 / 无商业实质** → 用**账面价值**

### 各章节对照

|         章节          |  交易性质  |     计价基础      |     差额去向      |
| :-----------------: | :----: | :-----------: | :-----------: |
|      **正常采购**       |  独立交易  |  公允价值（实际支付）   |      无差额      |
| **非货币性资产交换（有商业实质）** |  等价交换  | **换出资产公允价值**  | 资产处置损益/投资收益等  |
| **非货币性资产交换（无商业实质）** |  非独立   | **换出资产账面价值**  |    **无差额**    |
|    **债务重组（债权人）**    |  债务清偿  | **放弃债权的公允价值** |     投资收益      |
|  **非同控企业合并（购买法）**   |  购买交易  | **付出对价的公允价值** |   商誉或营业外收入    |
|  **同控企业合并（权益结合法）**  | 集团内部重组 |   **账面价值**    |     资本公积      |
|      **承租人租赁**      |  融资行为  |   **租金现值**    | 未确认融资费用（备抵科目） |

## 🔄 各章节分录模板速查

### 1️⃣ 非货币性资产交换（有商业实质）

```
借：换入资产                ← 按换出资产公允价 + 补价
    应交税费——应交增值税（进项税额）
    贷：固定资产清理/原材料/长投  ← 换出资产的账面价值
        银行存款                     ← 支付的补价
        应交税费——应交增值税（销项税额）
        资产处置损益/投资收益       ← 公允-账面（差额在贷方是收益）
```

> 详见 [[study/笔记-高财/整理-非货币性资产交换|非货币性资产交换笔记]]

### 2️⃣ 债务重组（债权人以资产抵债）

```
借：换入资产（存货/固资等）  ← 放弃债权的公允价值 - 进项税
    应交税费——进项税
    投资收益                   ← 差额损失（放弃债权公允 > 换入资产+进项税）
    贷：应收账款                  ← 放弃的债权账面价值
```

> 详见 [[第七章-债务重组|债务重组笔记]]

---

### 3️⃣ 非同控企业合并

```
借：长期股权投资             ← 付出对价的公允价值
    贷：银行存款                  ← 付出现金
        股本/资本公积             ← 发行股票的公允价值
        商誉                      ← 合并成本 > 可辨认净资产公允份额
```

> 详见 [[study/笔记-高财/高级财务会计-企业合并|企业合并笔记]]

### 4️⃣ 同控企业合并

```
借：长期股权投资             ← 被合并方净资产账面价值份额
    贷：股本                      ← 发行股票面值
        资本公积                  ← 差额（或借方先冲资本公积）
```

> ⚠️ 同控下 **不产生商誉**，差额走资本公积。还要 **恢复留存收益**！
>

### 5️⃣ 承租人租赁（初始计量）

```
借：使用权资产               ← 租金现值 + 初始直接费用
    租赁负债——未确认融资费用   ← 差额（未来利息）
    贷：租赁负债——租赁付款额       ← 租金总额（面值）
        银行存款                   ← 初始直接费用
```

```
      贷/借：财务费用——汇兑差额   ← 差额进当期损益
```

## 🎯 统一记忆：三问法

每次遇到要写分录的题，问自己三个问题：

| 问题 | 答案 | 对应的分录 |
|:----:|:----:|:----------:|
| **① 我得到什么？** | 新资产 | **借记**这个资产 |
| **② 我付出什么？** | 旧资产/现金/股票/债权 | **贷记**这些项目 |
| **③ 付出的价值怎么算？** | 独立交易→**公允**<br>内部交易→**账面** | 公允≠账面时，差额走损益 |

### 举例验证

**例子：甲公司用设备（账面150万、公允180万）+ 补价20万换入原材料，有商业实质**

```
三问法：
① 得到 → 原材料          → 借：原材料
② 付出 → 旧设备+补价     → 贷：固定资产清理（设备账面150）
                             贷：银行存款（补价20）
③ 有商业实质→用公允价值  → 设备公允180 ≠ 账面150 → 差额30万走资产处置损益

原材料入账价值 = 设备公允180 + 补价20 = 200万

借：原材料                200万
    贷：固定资产清理           150万（设备账面）
        银行存款               20万
        资产处置损益           30万
```

## 💡 高频易错点提示

| 场景 | 易错点 | 正确做法 |
|:----:|:------:|:---------|
| 非货币性交换换出**存货** | 差额记"资产处置损益" | 确认**主营业务收入**并结转**主营业务成本** |
| 非货币性交换换出**长投** | 忘了转出其他综合收益 | 借记**其他综合收益**，贷记**投资收益** |
| 非货币性交换**无商业实质** | 仍按公允价值入账 | 按**账面价值**入账，**无差额** |
| 债务重组收到资产 | 按收到资产的公允价值入账 | 按**放弃债权的公允价值**入账 |
| 同控合并 | 差额记"商誉" | 差额走**资本公积**，不产生商誉 |
| 租赁初始计量 | 忘记折现 | 租赁负债按**现值**入账，总额与现值的差额为未确认融资费用 |
