## 一、租赁的识别（选择题考点）

### 1.1 什么是租赁？

**大白话**：你付钱，我用东西给你用一段时间，到期还我。

**专业定义**：在一定期间内，出租人将资产的使用权让与承租人，以换取对价的合同。

### 1.2 租赁三要素（必须同时满足）

```
① 存在一定期间       → 租多久？
② 存在已识别资产     → 具体哪台设备/车辆/生产线？
③ 控制资产使用权     → 你能决定怎么用、获得全部经济利益
```

### 1.3 已识别资产的判断

| 条件               | 说明                      |
| ---------------- | ----------------------- |
| 合同明确指定或隐性指定      | 有具体编号或只有这台可用            |
| ❌ 供应方有**实质性替换权** | 能随时换一台且换了对它划算 → 不算已识别资产 |

### 1.4 控制使用权的判断（两条都要满足）

| 条件                 | 大白话            |
| ------------------ | -------------- |
| ① 有权获得**几乎全部经济利益** | 产品、节省的成本都归你    |
| ② 有权**主导资产使用**     | 你决定何时用、用在哪、怎么用 |

## 二、租赁的分拆与合并

### 2.1 分拆

合同中同时包含租赁部分和非租赁部分（如维保服务）：

| 角色 | 处理 |
|------|------|
| **承租人** | **可选择**分拆或不分拆。不分拆 → 全部当租赁处理 |
| **出租人** | **必须**分拆，按各自准则处理 |

### 2.2 合并

多份合同同时满足以下条件之一 → 合并为一份处理：
1. 基于总体商业目的，构成一揽子交易
2. 某份合同对价取决于其他合同的定价或履行
3. 多份合同合起来构成一项单独租赁

## 三、租赁的分类（仅出租人需要）⭐

> ⚠️ **承租人不需要区分融资租赁和经营租赁**，承租人对所有租赁（除豁免外）都确认使用权资产和租赁负债。

### 融资租赁的五条判定标准（满足任意一条即可）

| 标准                         | 大白话       |
| -------------------------- | --------- |
| ① 租期满时**所有权转移**给承租人        | 到期东西归你    |
| ② 有**低价购买选择权**，低到几乎肯定行使    | 到期可以白菜价买下 |
| ③ **租赁期 ≥ 资产剩余使用寿命的75%**   | 你租了它大半辈子  |
| ④ **租赁收款额现值 ≥ 资产公允价值的90%** | 租金差不多够买它了 |
| ⑤ 资产**性质特殊**，只有你能用         | 定做的，别人用不了 |

## 四、核心概念梳理

| 概念          | 对应谁   | 大白话                      |
| ----------- | ----- | ------------------------ |
| **租赁付款额**   | 承租人→付 | 承租人承诺未来要付的钱（固定+可确定的部分）   |
| **租赁收款额**   | 出租人→收 | 出租人预计能收到的钱（含担保余值）        |
| **使用权资产**   | 承租人   | 租来的东西，在账上当成自己的资产记        |
| **租赁负债**    | 承租人   | 欠着的租金，确认为一项负债            |
| **未确认融资费用** | 承租人   | 利息部分，以后分期摊到财务费用          |
| **担保余值**    | 双方    | 保证租赁到期时资产至少值多少钱          |
| **初始直接费用**  | 双方    | 为达成租赁发生的增量成本（佣金、印花税等）    |
| **租赁内含利率**  | 出租人计算 | 使租赁投资净额=资产公允价值+初始直接费用的利率 |

### 租赁付款额的构成

```
① 固定付款额（扣除租赁激励）
② 取决于指数或比率的可变租赁付款额（如挂钩CPI）✅ 计入
③ 购买选择权的行权价格（如果合理确定会行使）
④ 终止租赁选择权需付的款项
⑤ 担保余值预计应支付的款项
❌ 基于销售收入的可变租赁付款额 — 不计入，发生时直接进损益
```

## 五、承租人的会计处理 ⭐⭐⭐（大题核心！）

### 5.1 初始计量

#### 使用权资产的入账价值

```
使用权资产成本 = 
  ① 租赁负债的初始计量金额（租金现值）
  + ② 租赁期开始日之前支付的租金（扣除租赁激励）
  + ③ 初始直接费用（佣金、印花税等）
  + ④ 拆卸、移除、复原的预计成本现值
```

#### 租赁负债的初始计量

```
租赁负债 = 租赁期开始日尚未支付的租赁付款额的现值

折现率选择：
  优先 → 租赁内含利率
  否则 → 承租人增量借款利率
```

#### 初始计量分录模板

```
借：使用权资产                    （现值 + 初始直接费用 + …）
    租赁负债——未确认融资费用       （差额 = 利息总额）
    贷：租赁负债——租赁付款额        （未来租金总额）
        银行存款                    （初始直接费用）
```

### 5.2 后续计量（三步走）

#### ① 付租金 + 确认利息费用（实际利率法）

```
支付租金：
借：租赁负债——租赁付款额          （还本金部分）
    贷：银行存款

确认利息费用：
借：财务费用                      （年初本金余额 × 折现率）
    贷：租赁负债——未确认融资费用
```

**规律**：利息逐年递减，本金偿还逐年递增（摊余成本计量的典型运用）。

#### ② 计提使用权资产折旧

```
折旧期间：
  能取得所有权 → 资产剩余使用寿命
  不能取得所有权 → 租赁期与剩余使用寿命孰短

借：制造费用/管理费用/销售费用等
    贷：使用权资产累计折旧
```

#### ③ 可变租赁付款额

```
基于收入的可变租赁付款额 → 发生时：
  借：销售费用/生产成本
      贷：银行存款/其他应付款
```

### 5.3 租赁期届满的处理

| 情形     | 处理            |
| ------ | ------------- |
| **返还** | 冲销使用权资产和累计折旧  |
| **续租** | 视同租赁一直存在继续处理  |
| **留购** | 支付购买价款，转入固定资产 |

### 5.4 租赁负债的重新计量（四种情形）

```
① 实质固定付款额发生变动
② 担保余值预计应付金额变动
③ 用于确定租赁付款额的指数或比率变动
④ 购买/续租/终止选择权的评估结果或实际行使情况变化

增加时：借：使用权资产  贷：租赁负债
减少时：借：租赁负债  贷：使用权资产
（使用权资产调至零后仍有差额 → 计入当期损益）
```

## 六、短期租赁和低价值资产租赁（选择题重点）

| 类型          | 条件                           | 处理方式                          |
| ----------- | ---------------------------- | ----------------------------- |
| **短期租赁**    | 租赁期开始日 ≤ 12个月，**不含购买选择权**    | 可选择豁免，不确认使用权资产和租赁负债，直线法确认租金费用 |
| **低价值资产租赁** | 单项资产全新时价值较低（与承租人规模无关，绝对金额判断） | 同上，可豁免                        |

> ⚠️ 承租人转租或预期转租的 → 原租赁不属于低价值资产租赁

## 七、出租人的会计处理

### 7.1 经营租赁（出租人）

```
确认租金收入（直线法）：
借：银行存款/应收账款
    贷：租赁收入

计提折旧：
借：其他业务成本
    贷：累计折旧
```

### 7.2 融资租赁（出租人）

**实质**：相当于出租人"卖"资产，钱分期收。

**初始计量**：

```
租赁投资净额 = 未担保余值现值 + 租赁收款额现值
             = 租赁资产公允价值 + 初始直接费用

借：应收融资租赁款——租赁收款额     （租赁收款总额）
    贷：融资租赁资产                 （资产账面价值）
        应收融资租赁款——未实现融资收益  （差额=利息）
        资产处置损益                 （公允-账面，如有）
```

**后续计量**（收取租金时）：

```
借：银行存款
    贷：应收融资租赁款——租赁收款额

借：应收融资租赁款——未实现融资收益
    贷：租赁收入                    （利息收入，实际利率法）
```

## 八、售后租回（了解即可）

**本质**：一种特殊的融资活动。先卖资产 → 收到钱 → 立刻租回继续使用。

资产转让**属于销售**时：
- 承租人：按资产账面价值中与租回使用权有关的部分计量使用权资产
- 仅就转让至出租人的权利确认相关利得或损失

资产转让**不属于销售**时：
- 承租人：继续确认被转让资产 + 确认一项金融负债

## 九、列报（了解即可）

| 项目 | 承租人 | 出租人 |
|------|--------|--------|
| **资产负债表** | 使用权资产（单独列示）、租赁负债（分流动/非流动） | 经营租赁资产、应收融资租赁款 |
| **利润表** | 利息费用（财务费用） + 折旧费（分别列示） | 租赁收入、利息收入 |
| **现金流量表** | 偿还租赁负债本息 → 筹资活动；短期/低价值租金 → 经营活动 | 收取租金 → 经营活动 |

## 十、易错点汇总 ⚠️

| 序号  | 易错点                     | 正确结论                    |
| :-: | ----------------------- | ----------------------- |
|  ①  | 承租人要不要区分融资/经营租赁？        | ❌ **不需要**，都确认使用权资产+租赁负债 |
|  ②  | 基于销售收入的可变租赁付款额算不算租赁付款额？ | ❌ **不算**，发生时直接进销售费用     |
|  ③  | 短期租赁豁免的条件？              | 租赁期 ≤ 12个月 **且不含购买选择权** |
|  ④  | 低价值资产租赁的判定标准？           | 只看资产**绝对价值**，不看承租人规模    |
|  ⑤  | 折旧期间怎么定？（不能取得所有权时）      | **租赁期**与**剩余使用寿命**孰短    |
|  ⑥  | 使用权资产减值后能否转回？           | ❌ **不能转回**              |
|  ⑦  | 折现率优先选哪个？               | **租赁内含利率** > 承租人增量借款利率  |
|  ⑧  | 初始直接费用记哪里？              | 计入**使用权资产**入账价值         |

## 十二、承租人与出租人对比总表

| 事项 | 承租人 | 出租人（融资租赁） | 出租人（经营租赁） |
|:----:|:------:|:-----------------:|:-----------------:|
| 初始确认 | 使用权资产 + 租赁负债 | 应收融资租赁款（终止确认资产） | 继续确认租赁资产 |
| 每期处理 | 付租金 + 利息费用 + 折旧 | 收租金 + 确认利息收入 | 收租金 + 提折旧 |
| 利息计算 | 实际利率法 | 实际利率法 | 不适用 |
| 到期处理 | 返还/续租/留购 | 收回/续租/留购 | 收回 |

---

> 💡 **复习建议**：承租人会计处理是绝对核心，必须熟练掌握初始计量和后续计量的分录。选择题重点掌握租赁识别判断、融资租赁五条标准、短期租赁和低价值资产租赁的豁免条件。出租人部分重点掌握融资租赁和经营租赁的分类判断。
