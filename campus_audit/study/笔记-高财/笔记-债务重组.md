# 第七章 债务重组 —— 考点全景总结

> 📌 **考纲定位**：「掌握债权人和债务人以资产清偿债务时各自的会计处理、债务重组损益的计算」

> 🚨 **考前急救 · 选择题必考点速查（3分钟看完）** 🚨
> 
> ### 🔥 最常考的四组对比
> 
> | 对比 | 结论 | 陷阱 |
> |:----:|:----|:----:|
> | **债权人入账用谁的公允？** | ✅ **放弃债权的**公允价值 | ❌ 不是换入资产公允！ |
> | **债权人差额记哪？** | ✅ **投资收益**（永远是投资收益） | ❌ 不是资产减值损失 |
> | **债务人差额记哪？（非金融资产抵债）** | ✅ **其他收益——债务重组收益** | ❌ 不是资产处置损益 |
> | **债务人差额记哪？（金融资产抵债）** | ✅ **投资收益** | ❌ 不是其他收益 |
> 
> ### 🔥 五个"是/不是"判断
> 
> | 判断 | 答案 |
> |:----|:----:|
> | 新准则要求债务人"财务困难"吗？ | ❌ **不需要**（旧准则才要） |
> | 债务人用存货抵债要确认收入吗？ | ❌ **不需要**，直接按账面转，差额进其他收益 |
> | 债务人要区分"资产处置损益"和"债务重组损益"吗？ | ❌ **不区分**，合并计入一个科目 |
> | 关联方债务重组如果构成权益性交易，确认损益吗？ | ❌ **不确认**，按权益性分配处理 |
> | 债务人差额 = 债务账面 - 资产**公允**？ | ❌ **是资产账面**（不是公允） |
> 
> ### 🔥 速记口诀
> 
> ```
> 债权人：放弃债权公允入账，差额全走投资收益
> 债务人：资产按账面转出，差额全走其他收益（非金融）/投资收益（金融）
> 对比交换：交换用换出公允，重组用放弃债权公允 ← ⚠️ 必考！
> ```
>

## 一、债务重组的定义

### 新旧准则对比

| 对比项 | 旧准则（2006版） | 新准则（现行） |
|:------:|:----------------:|:--------------:|
| 前提条件 | 债务人**发生财务困难** + 债权人**作出让步** | 不要求财务困难，也不要求让步 |
| 定义 | 债权人按照协议或裁定**作出让步**的事项 | 在不改变交易对手方的情况下，就清偿债务的**时间、金额或方式**重新达成协议的交易 |

**大白话**：以前必须是要倒闭了+债权人割肉才算债务重组；现在只要是**重新谈了个还钱方案**（不管原因）就算。

### 适用范围

| 属于债务重组的债权债务 | 不属于债务重组的 |
|:----------------------:|:----------------:|
| 金融工具准则规范的债权债务（应收账款、贷款、应付账款等） | 合同资产、合同负债、预计负债 |
| **租赁应收款和租赁应付款** | — |

### ⚠️ 债务重组构成权益性交易

如果债权人和债务人是**关联方**（比如母公司免了子公司的债），且以股东身份进行债务重组 → **不确认债务重组损益**，按权益性交易处理。

```
判断标准：
① 债权人直接或间接对债务人持股，且以股东身份进行重组
② 债务重组前后均受同一方最终控制，且实质是权益性分配或投入
```

## 二、债务重组的方式（四种）

```
① 以资产清偿债务       → 用东西抵债
② 将债务转为权益工具    → 债转股
③ 修改其他条款         → 展期/降息/减本金
④ 组合方式             → 以上三种混合
```

---

## 三、核心会计处理 ⭐⭐⭐

### 3.1 债权人会计处理总原则

```
受让非现金资产的成本 = 放弃债权的公允价值 + 可直接归属的税金/运费等
放弃债权的公允价值与账面价值的差额 → 投资收益
```

> ⚠️ **注意**：跟非货币性资产交换不同！非货币性交换用换出资产的公允价值，债务重组用**放弃债权的公允价值**！

### 3.2 债务人会计处理总原则

```
以非金融资产清偿债务：
所清偿债务账面价值与转让资产账面价值的差额 → 计入「其他收益——债务重组收益」

以金融资产清偿债务：
所清偿债务账面价值与转让资产账面价值的差额 → 计入「投资收益」
```

> ⚠️ **债务人不需要区分资产处置损益和债务重组损益**，合并计入一个科目！

## 四、以资产清偿债务（最常考）⭐⭐⭐

### 4.1 债权人处理

```
借：换入资产（存货/固定资产/无形资产等）
          （放弃债权的公允价值 + 相关税费）
    坏账准备                     （已提的坏账准备）
    投资收益                     （差额损失）
    贷：应收账款/债权投资等           （账面余额）
        银行存款                     （支付的运费等）
```

> **公式**：
> ```
> 换入资产入账价值 = 放弃债权的公允价值 + 相关税费
> 投资收益 = 放弃债权的账面价值 - 放弃债权的公允价值 - 相关税费
> ```

### 4.2 债务人处理

```
① 资产转清理（固资/无形资产）：
借：固定资产清理
    累计折旧
    固定资产减值准备
    贷：固定资产

② 清偿债务：
借：应付账款                     （债务账面价值）
    贷：固定资产清理                  （资产账面价值）
        其他收益——债务重组收益        （差额）
```

> **公式**：
> ```
> 债务重组收益 = 债务账面价值 - 转让资产账面价值
> ```

---

### 📌 例题精讲（例1）

> 甲公司应收乙公司95万元。乙公司用一项固定资产抵债。固定资产原值100万，累计折旧10万，已提减值准备2万，账面价值88万。甲公司已提坏账准备7万，支付评估费4万。当日应收账款公允价值87万。

**债权人（甲公司）：**

```
换入固定资产成本 = 放弃债权公允价值87万 + 评估费4万 = 91万
投资收益 = 95万（账面） - 87万（公允） - 4万 = 4万（损失在贷方是4万？）

等一下，准确来说是：
放弃债权账面价值 = 95万
放弃债权公允价值 = 87万
差额 = 8万（损失）
坏账准备 = 7万
实际计入投资收益 = 8万 - 7万 = 1万（其实等于（账面余额95-坏账7）-公允87-费用4... 不对）
```

**准确的计算：**

```
固定资产入账成本 = 放弃债权公允价值87万 + 评估费4万 = 91万

借：固定资产             91万   (87+4)
    坏账准备              7万
    投资收益              1万   ← 倒挤
    贷：应收账款              95万
        银行存款               4万
```

**验算：借方91+7+1=99，贷方95+4=99 ✅**

> 投资收益1万 = 放弃债权账面价值(95-7=88) - 放弃债权公允价值(87) + 坏账准备(7) - 运输费... 
> 简单理解：实际损失 = 收不回来的部分

**债务人（乙公司）：**

```
① 固定资产清理
借：固定资产清理         88万
    累计折旧             10万
    固定资产减值准备      2万
    贷：固定资产              100万

② 清偿债务
借：应付账款             95万
    贷：固定资产清理           88万
        其他收益——债务重组收益    7万
```

> 债务重组收益 = 95万 - 88万 = **7万** ✅

## 五、将债务转为权益工具（债转股）

### 5.1 债权人处理

```
借：长期股权投资/交易性金融资产等
          （放弃债权的公允价值 + 相关税费）
    坏账准备
    投资收益              （差额损失）
    贷：应收账款/债权投资等   （账面余额）
        银行存款              （手续费等）
```

### 5.2 债务人处理

```
权益工具按公允价值计量；公允不能可靠计量的，按所清偿债务的公允价值计量。
差额 → 投资收益（不是其他收益！因为涉及的是金融工具）

借：应付账款等              （债务账面价值）
    贷：股本/实收资本             （面值）
        资本公积——股本溢价        （权益工具公允-面值）
        银行存款                  （发行费用）
        投资收益                  （差额）
```

## 六、修改其他条款

### 判断是否构成实质性修改

```
如果重组后现金流量的现值与原债务剩余期间现金流量现值的差异超过10%
→ 构成实质性修改 → 终止确认原债务，确认新金融负债

计算公式：
|新现值 - 原现值| ÷ 原现值 > 10%
```

| 情形 | 处理 |
|:----:|:------|
| **实质性修改** | 终止确认原债权/债务，按公允价值确认新金融资产/负债，差额进**投资收益** |
| **非实质性修改** | 不终止确认，重新计算账面余额，差额调整**投资收益** |

## 七、组合方式

### 7.1 债权人处理

```
① 金融资产：按公允价值确认
② 非金融资产：按放弃债权公允价值扣除金融资产后的净额，按各项资产公允价值比例分配
③ 差额 → 投资收益
```

### 7.2 债务人处理

```
① 金融资产清偿部分 → 投资收益
② 非金融资产清偿部分 → 其他收益——债务重组收益
③ 修改其他条款部分 → 按修改条款规则处理
```

## 八、债权人 vs 债务人 对比总表

| 对比项 | 债权人 | 债务人 |
|:------:|:------:|:------:|
| **入账价值** | **放弃债权的公允价值** + 相关税费 | 不适用（直接清偿债务） |
| **差额科目（非金融资产）** | **投资收益** | **其他收益——债务重组收益** |
| **差额科目（金融资产）** | **投资收益** | **投资收益** |
| **是否区分资产处置损益** | 不区分 | **不区分**，合并计入一个科目 |
| **坏账准备** | 结转已计提的坏账 | 不涉及 |
| **公允价值变动的转出** | 其他综合收益→投资收益 | 其他综合收益→投资收益或留存收益 |

---

## ⚠️ 高频易错点

| 序号 | 易错点 | 正确做法 |
|:----:|:------:|:---------|
| ① | 债权人换入资产按**换入资产公允价值**入账 | ✅ 按**放弃债权的公允价值**入账 |
| ② | 债务人区分"资产处置损益"和"债务重组损益" | ✅ **不区分**，合并计入**其他收益——债务重组收益** |
| ③ | 债权人的差额记"资产减值损失" | ✅ 记**投资收益** |
| ④ | 构成权益性交易的债务重组确认损益 | ❌ **不确认损益**，按权益性交易处理 |
| ⑤ | 债务人以存货抵债确认收入 | ✅ 不需要确认销售收入，直接按账面价值转出，差额进**其他收益——债务重组收益** |
| ⑥ | 债务重组收益=债务账面-资产公允 | ❌ 是**债务账面-资产账面**（不是公允） |

---

## 九、债务重组 vs 非货币性资产交换 对比

| 对比项 | 非货币性资产交换 | 债务重组 |
|:------:|:----------------:|:--------:|
| **交易目的** | 等价交换（以物换物） | 偿债（以物抵债） |
| **换入资产计量基础** | **换出资产公允价值**（或账面） | **放弃债权的公允价值** |
| **损益科目（非金融资产）** | 资产处置损益/主营业务收入 | **其他收益——债务重组收益**（债务人）/ **投资收益**（债权人） |
| **补价限制** | 补价比例 ≤ 25% | 无补价比例限制 |

---

> 💡 **复习建议**：债务重组以**以资产清偿债务**为考试重点，核心是债权人按放弃债权的公允价值入账，差额进投资收益；债务人按资产账面价值转出，差额进其他收益——债务重组收益。注意与非货币性资产交换区分，两者容易混淆。债转股和修改条款了解即可。
