## 一、核心概念：同控 vs 非同控（必考判断）⭐

这是本章**一切会计处理的起点**——先判断属于哪种合并，再用对应的规则做账。

| 比较维度        | 同一控制下企业合并（同控）                                    | 非同一控制下企业合并（非同控）                    |
| ----------- | ------------------------------------------------ | ---------------------------------- |
| **通俗理解**    | 老爸把大儿子和小儿子组到一起（合并前后都受同一方控制）                      | 陌生人之间的买卖（合并前后不受同一方控制）              |
| **判断标准**    | 参与合并的企业在**合并前后**均受**同一方或相同多方最终控制**，且该控制**并非暂时性** | 参与合并的各方在合并前后**不受同一方或相同多方最终控制**     |
| **计量基础**    | **账面价值**原则                                       | **公允价值**原则（购买法）                    |
| **是否产生商誉？** | ❌ **不产生商誉**                                      | ✅ **产生商誉**                         |
| **差额去哪？**   | 差额 → **资本公积**（不够冲的冲留存收益）                         | 正差额 → **商誉**；负差额 → **营业外收入**（廉价购买） |

> **课件·单选题（2017年）**：收购受同一母公司控制的乙公司60%股权 → 同控 → 恢复留存收益 → 影响合并所有者权益变动表留存收益的年初金额。

> **课件·多选题（2021年）**：
> 
> - A ✅ 企业内部产生的商誉不应确认为资产
> - B ❌ 商誉不是无形资产（不具有可辨认性）
> - C ✅ 同控下差额不确认为商誉，进资本公积
> - D ✅ 非同控下支付对价 < 可辨认净资产公允价值份额 → 差额进当期损益（营业外收入）

## 二、同一控制下企业合并（同控）的处理

### 2.1 三个核心原则

```
① 以账面价值为基础：入账不看公允价值，只看账面价值
② 不产生商誉：差额走资本公积
③ 必须恢复留存收益：同控最容易被遗忘的一步！⚠️
```

### 2.2 账务处理步骤

> **课件·案例题1**：A、B均为P的子公司，A发行1,500万股（面值1元）取得B公司100%股权。

| 步骤         | 做什么                       | 具体计算                                           |
| ---------- | ------------------------- | ---------------------------------------------- |
| ① 确认长期股权投资 | 按最终控制方合并报表中净资产的**账面价值份额** | 长期股权投资 = B公司净资产账面价值 5,000万 × 100% = **5,000万** |
| ② 确认对价     | 按发行股份面值（或支付资产账面价值）        | 贷：股本 1,500万                                    |
| ③ 差额       | 倒挤                        | 借方差额优先冲资本公积：5,000 - 1,500 = 3,500万 → 资本公积      |

```
借：长期股权投资              5,000万
    贷：股本                        1,500万
        资本公积——股本溢价          3,500万
```

### 2.3 ⚠️ 恢复留存收益（同控最易遗忘的步骤）

> **考纲原文**：「同控情况下，大家最容易忘掉'恢复留存收益'这一步，请务必重点关注。」

**逻辑**：合并前B公司自己赚的留存收益（盈余公积1,000万 + 未分配利润2,000万 = 3,000万），在[[第27章 合并财务报表（全）|合并报表]]层面要"还原回来"。因为站在最终控制方角度看，这些留存收益本来就存在。

```
借：资本公积              3,000万  ← 从资本公积里挪
    贷：盈余公积                1,000万
        未分配利润              2,000万
```

⚠️ **限制**：只能恢复资本公积（股本溢价）足够覆盖的部分。如果资本溢价余额不够，有多少恢复多少。

> 课件原文：A公司确认长期股权投资后，资本公积余额为 6,000万（2,500 + 3,500），其中股本溢价 4,500万 → 足够恢复3,000万留存收益。

## 三、非同一控制下企业合并（非同控）的处理 ⭐

### 3.1 四个核心步骤

```
① 确定购买方（掏钱/发股的那个）
② 确定购买日（实际取得控制权的日期）
③ 计算合并成本（付出了什么 = 公允价值合计）
④ 计算商誉（合并成本 - 可辨认净资产公允价值份额）
```

### 3.2 商誉的计算（计算题核心）

```
商誉 = 合并成本 - 购买日可辨认净资产公允价值 × 持股比例
```

|结果|处理|
|---|---|
|**正商誉**（合并成本 > 份额）|确认为**商誉**（资产），每年减值测试|
|**负商誉**（合并成本 < 份额）|计入**营业外收入**（廉价购买利得）|

> **课件·案例题4**：P公司发行1,000万股（面值1元，市价8.75元，即合并成本 = 1,000 × 8.75 = 8,750万），取得S公司70%股权。S公司可辨认净资产公允价值 = 10,850万。
> 
> **商誉 = 8,750 - 10,850 × 70% = 8,750 - 7,595 = 1,155万** ✅

### 3.3 购买日的调整分录（非同控特有的公允价值调整）

非同控下，需要把子公司的资产负债从**账面价值调整为公允价值**（此调整在[[第27章 合并财务报表（全）|合并财务报表]]编报时持续影响后续各期）：

> **课件·案例题4**：S公司存货账面255万 → 公允450万（+195），固定资产账面3,000万 → 公允5,500万（+2,500），无形资产账面500万 → 公允1,500万（+1,000），长期股权投资账面2,150万 → 公允3,800万（+1,650），合计增值 = 5,345万。

```
借：存货                    195万
    长期股权投资            1,650万
    固定资产                2,500万
    无形资产                1,000万
    贷：资本公积                5,345万   ← 子公司净资产公允价值增加
```

### 3.4 购买日的抵销分录（母公司的长投 vs 子公司的权益）

```
借：股本                    2,500万   ← 子公司账面
    资本公积                6,845万   ← 子公司账面1,500 + 公允调增5,345
    盈余公积                  500万
    未分配利润              1,005万
    商誉                    1,155万   ← 挤出来的
    贷：长期股权投资            8,750万   ← 母公司付出的代价
        少数股东权益            3,255万   ← 10,850 × 30%（其他股东占的份额）
```

### 3.5 考虑所得税时（课件·案例题5）

有递延所得税时，公允价值调增5,345万 → 产生应纳税暂时性差异 → 确认**递延所得税负债** = 5,345 × 25% = 1,336.25万：

```
借：资本公积                1,336.25万
    贷：递延所得税负债            1,336.25万
```

此时可辨认净资产公允价值 = 5,505 + 5,345 - 1,336.25 = 9,513.75万

商誉 = 8,750 - 9,513.75 × 70% = 8,750 - 6,659.625 = **2,090.38万**

> 📌 但考纲说「合并报表递延所得税的调整考试不涉及、不会考」，所以这里了解即可。

## 四、或有对价（业绩对赌）⭐⭐⭐

> 这是近年**高频考点**！课件里有4道案例题涉及。

### 4.1 什么是有对价？

企业合并时，卖方（原股东）承诺：如果被收购公司未来业绩达到/未达到某个目标，买方需多付钱或少付钱（或退回股份等）。

**分类**：

| 或有对价类型           | 会计分类           | 后续公允价值变动       |
| ---------------- | -------------- | -------------- |
| 以**现金**补偿（金额不固定） | **交易性金融负债/资产** | 计入**公允价值变动损益** |
| 以**固定数量股份**补偿    | **权益工具**       | 不做会计处理         |

> **课件·单选题（2020年）**：乙公司将以不固定股份数量返还甲公司 → 属于金融工具 → **交易性金融资产**。

### 4.2 购买日处理

购买日，按或有对价的**公允价值**计入合并成本。

> **课件·案例题8**：5,000万（固定）+ 税后净利润 × 2（可变）= 合并成本。购买日预计净利润1,200万 → 或有应付 = 1,200 × 2 = 2,400万 → 合并成本 = 5,000 + 2,400 = **7,400万**。

```
借：长期股权投资          7,400万
    贷：银行存款                5,000万
        交易性金融负债            2,400万
```

### 4.3 后续处理（实际业绩 ≠ 预期时）

> **课件·案例题8②**：实际净利润1,500万 → 应支付1,500 × 2 = 3,000万

```
借：交易性金融负债          2,400万   ← 冲掉原负债
    投资收益                  600万   ← 多付的 → 损失
    贷：银行存款                3,000万
```

> **课件·案例题8③**：实际净利润800万 → 应支付800 × 2 = 1,600万。同时原股东承诺不足1,200万部分无偿赠予100万股。

```
借：交易性金融负债          2,400万
    贷：银行存款                1,600万
        投资收益                  800万   ← 少付了 → 收益

借：股本                      100万   ← 收到赠予的股票并注销
    贷：资本公积                  100万
```

> **课件·案例题7（教材26-9）**：2×10年业绩不达标且预计未来也无法达标 → 确认或有对价公允价值2,000万。由于是购买日后超过12个月的新情况，**不调整合并成本**，而是计入当期损益：
> 
> ```
> 借：交易性金融资产          2,000万
>     贷：公允价值变动损益          2,000万
> ```

## 六、分步实现企业合并（2022年综合题·案例题13）

> 这个课件案例是**2022年真题**，虽然考纲没单独列，但课件有完整内容，建议了解基本框架。

**核心逻辑**：

1. 先买少数股权（权益法核算）→ 再买多数股权实现控制
2. 合并成本 = 新增对价公允价值 + 原持有股权在购买日的公允价值
3. 原持有股权按购买日公允价值**重新计量**，差价进投资损益

**商誉计算**：

> 案例题13：合并成本 = 750万股 × 10元 + 原25%公允价值2,500万 = 10,000万；可辨认净资产公允价值 = 8,670万；商誉 = 10,000 - 8,670 = **1,330万**。

## 七、选择题高频易错点汇总

|序号|考点|正确结论|
|---|---|---|
|①|同控下差额去哪？|**资本公积**（资本溢价），不够冲的依次冲留存收益|
|②|同控下产生商誉吗？|❌ **不产生商誉**|
|③|非同控下合并成本 < 可辨认净资产公允价值份额？|差额计入**营业外收入**（当期损益）|
|④|同控下最容易忘什么？|⚠️ **恢复留存收益**！|
|⑤|非同控下子公司资产负债按什么计量？|**公允价值**|
|⑥|同控下子公司资产负债按什么计量？|**账面价值**|
|⑦|商誉是无形资产吗？|❌ 不是。商誉不具有可辨认性|
|⑧|或有对价（现金补偿）后续变动？|计入**公允价值变动损益**，不追溯调整合并成本（购买日后超12个月的新情况）|
|⑨|交易费用（非同控）？|计入**管理费用**（不是长期股权投资成本！）|
|⑩|发行股票相关佣金？|冲减**资本公积——股本溢价**|

> **课件·案例题13**：进一步购买75%股权时，支付税费700万 → 管理费用；发行股票佣金800万 → 冲资本公积。

## 八、同控 vs 非同控 处理流程对比（一图搞定）

```
                收到企业合并消息
                      │
          ┌───────────┴───────────┐
          ▼                       ▼
    合并前后受同一方控制？     不受同一方控制
          │                       │
    【同一控制】              【非同一控制】
          │                       │
    长投 = 账面价值份额       长投 = 合并成本
    （不产生商誉）             （公允价值）
          │                       │
    差额 → 资本公积          合并成本 > 份额 → 商誉
          │                合并成本 < 份额 → 营业外收入
          │                       │
    ⚠️ 恢复留存收益！        调整子公司资产/负债
    （同控最易忘）            从账面价值 → 公允价值
```
