## 一、所得税（5题）

---

### 第1题

> 企业持有的交易性金融资产，资产负债表日公允价值大于账面价值，产生应纳税暂时性差异，应确认递延所得税负债。

**我的答案：** ____　　**正确答案：** ✅ 对

**解析：**

```
交易性金融资产：
  账面价值 = 公允价值（假设120万）  
  计税基础 = 初始取得成本（100万）

账面(120万) > 计税基础(100万)
→ 未来出售时，税法允许扣除的成本只有100万
→ 应纳税所得额会多20万 → 未来多交税
→ 应纳税暂时性差异 → 递延所得税负债 ✅

分录：
借：所得税费用              5万（20×25%）
    贷：递延所得税负债           5万
```

---

### 第2题

> 企业计提的应收账款坏账准备，税法不允许税前扣除，产生应纳税暂时性差异，应确认递延所得税负债。

**我的答案：** ____　　**正确答案：** ❌ 错

**解析：**

```
应收账款：
  账面价值 = 原值 - 坏账准备（假设原值200万，坏账20万，账面=180万）
  计税基础 = 原值 200万（税法不承认坏账准备）

账面(180万) < 计税基础(200万)
→ 未来实际发生坏账时，税法允许扣除
→ 未来少交税 → 可抵扣暂时性差异
→ 确认递延所得税资产，不是递延所得税负债！

正确分录：
借：递延所得税资产          5万（20×25%）
    贷：所得税费用               5万

⚠️ 易混：资产账面<计税基础 → 可抵扣 → 递延所得税资产
        资产账面>计税基础 → 应纳税 → 递延所得税负债
```

---

### 第3题

> 企业发生的超过税法扣除标准的业务招待费，属于可抵扣暂时性差异，应确认递延所得税资产。

**我的答案：** ____　　**正确答案：** ❌ 错

**解析：**

```
永久性差异 vs 暂时性差异的关键区别：

广告费超过部分   → 可以结转以后年度扣除 → 可抵扣暂时性差异 ✅
业务招待费超过部分 → ❌ 不能结转以后年度 → 永久性差异！

永久性差异的特点：影响当期所得税，但不产生递延所得税
暂时性差异的特点：当期不让扣，以后可以扣，产生递延所得税
```

| 项目 | 超过标准部分处理 | 差异类型 |
|:----:|:--------------:|:--------:|
| 广告费 | 结转以后年度扣除 | 可抵扣暂时性差异 ✅ |
| 职工教育经费 | 结转以后年度扣除 | 可抵扣暂时性差异 ✅ |
| **业务招待费** | **不能结转** | **永久性差异** ❌ |
| 行政罚款 | 永远不能扣除 | 永久性差异 ❌ |

---

### 第4题

> 使用寿命不确定的无形资产，会计上不摊销但每年进行减值测试，其计税基础为原值减去税法允许的摊销额，两者可能产生暂时性差异。

**我的答案：** ____　　**正确答案：** ✅ 对

**解析：**

```
例：无形资产原值500万，使用寿命不确定
  第1年末：未发生减值

  账面价值 = 500万（不减值就不变）
  计税基础 = 500 - 500÷10 = 450万（税法按10年摊销）

  账面(500) ≠ 计税基础(450) → 产生暂时性差异 ✅

  第2年末：发生减值100万

  账面价值 = 400万（500-100）
  计税基础 = 400万（500-500÷10×2）

  此时若账面<计税基础 → 可抵扣暂时性差异 → 递延所得税资产

⚠️ 关键：不摊销≠没有暂时性差异，因为：
  ① 税法**允许摊销** → 计税基础逐年减少
  ② 可能**计提减值** → 账面价值减少
```

---

### 第5题

> 企业确认的预计负债（如未决诉讼），计税基础等于账面价值，不产生暂时性差异。

**我的答案：** ____　　**正确答案：** ❌ 错

**解析：**

```
预计负债的计税基础公式：
  计税基础 = 账面价值 - 未来可税前扣除的金额

产品质量保证形成的预计负债：
  账面价值 = 100万
  未来实际发生保修时可扣除 = 100万
  计税基础 = 100 - 100 = 0
  账面(100) > 计税基础(0) → 可抵扣暂时性差异 ✅

未决诉讼形成的预计负债（假设税法规定实际支付时才扣除）：
  同上 → 可抵扣暂时性差异

但 ⚠️ 特殊情况：
  某些预计负债（如推迟缴纳的税款罚款），税法不允许扣除
  计税基础 = 账面价值 - 0 = 账面价值 → 无差异

结论：预计负债**通常**产生可抵扣暂时性差异，但不是绝对的，
      要看税法是否允许未来扣除。本题说"计税基础等于账面价值"
      是错的，因为大多数预计负债的计税基础 = 0。
```

## 二、外币折算（4题）

---

### 第6题

> 企业收到投资者投入的外币资本，应按合同约定的汇率折算为记账本位币入账。

**我的答案：** ____　　**正确答案：** ❌ 错

**解析：**

```
⚠️ 外币投资的**特殊规定**：

收到外币投资时 — 只能用**即期汇率**（收到当天的汇率），
不能用合同约定的汇率！

正确分录：
  借：银行存款——美元户   （美元金额 × 收到日即期汇率）
      贷：实收资本             （同上）

例：收到100万美元投资，合同约定汇率6.8，收到日即期汇率6.9

❌ 错误（用合同汇率）：
  借：银行存款——美元户    680万
      贷：实收资本              680万

✅ 正确（用即期汇率）：
  借：银行存款——美元户    690万
      贷：实收资本              690万

💡 为什么？
  实收资本是历史成本计量，一旦入账就不变了。
  合同汇率可能和实际收到时的汇率不一致，
  为了避免后续调整麻烦，直接用收到日的即期汇率最合理。
```

---

### 第7题

> 外币货币性项目（如应收账款、应付账款）期末按即期汇率调整产生的汇兑差额，应计入其他综合收益。

**我的答案：** ____　　**正确答案：** ❌ 错

**解析：**

```
外汇交易和报表折算的汇兑差额去向完全不同：

                场景                        差额去向
  ───────────────────────────────────────────────
  外币交易（应收账款/应付账款）期末调整 →  财务费用——汇兑差额 ✅
  外币报表折算（境外子公司）        →  其他综合收益 ✅
  其他权益工具投资（外币）汇率变动   →  其他综合收益 ✅
  其他债权投资（外币）汇兑差额       →  财务费用 ✅

分录示例（应收账款）：
  期初：应收账款100万美元，汇率6.8，记账680万
  期末：汇率变为6.9，应收账款应记为690万

  借：应收账款                  10万
      贷：财务费用——汇兑差额         10万 ✅（进当期损益）

💡 为什么应收账款调整进财务费用？
  因为这是日常经营活动中产生的汇率风险，
  直接影响当期利润。
  ⚠️ 而境外子公司报表折算差额进其他综合收益，
  是因为还没真正变现，避免利润剧烈波动。
```

---

### 第8题

> 以公允价值计量且其变动计入其他综合收益的外币非交易性权益工具投资，其汇兑差额应计入当期损益。

**我的答案：** ____　　**正确答案：** ❌ 错

**解析：**

```
⚠️ 这是最容易混淆的考点！

其他权益工具投资（外币）：
  既有公允价值变动，又有汇率变动
  
  但准则规定：**全部变动都计入其他综合收益**，不分拆！

例：买入10万美元的某公司股票（指定为其他权益工具投资）
  买入日汇率6.8，入账成本68万
  期末：股票市价12万美元，汇率6.9

  ✅ 全部公允价值变动（含汇率影响）→ 其他综合收益
  借：其他权益工具投资   14.8万（12×6.9-68）
      贷：其他综合收益         14.8万

对比：其他债权投资（外币）→ 汇兑差额进财务费用！
  
💡 速记：
  权益工具（股票）→ 全部进OCI
  债务工具（债券）→ 汇兑差额进损益
```

---

### 第9题

> 企业对境外子公司的外币财务报表进行折算时，资产负债表中的资产和负债项目应采用交易发生日的即期汇率折算。

**我的答案：** ____　　**正确答案：** ❌ 错

**解析：**

```
外币报表折算的汇率选择：

    报表项目                折算汇率
  ──────────────────────────────────
  资产负债表：
    资产和负债            **期末即期汇率** ✅
    所有者权益项目（除未分配利润）   **历史汇率** ✅
    未分配利润            轧差（倒挤）
  
  利润表：
    收入和费用            **交易发生日汇率** 
                         或 平均汇率（近似）
  
  折算差额：
    计入**其他综合收益** ✅

所以：
  ❌ 资产项目用交易发生日汇率 → 错！应用期末即期汇率
  ✅ 只有利润表的收入费用用交易发生日汇率（或平均汇率）

💡 口诀：
  "资产负债权用期末，权益除未分外历史，
   利润表用平均率，差额统统进OCI"
```

## 三、企业合并（5题）

---

### 第10题

> 同一控制下企业合并，合并方发生的审计、法律等中介费用应计入长期股权投资的初始投资成本。

**我的答案：** ____　　**正确答案：** ❌ 错

**解析：**

```
交易费用的处理（三类别）：

    费用类型              处理方式
  ─────────────────────────────────────
  中介费用（审计/法律/评估） → 计入**管理费用** ✅（无论同控还是非同控）
  发行股票的佣金手续费      → 冲减**资本公积——股本溢价**
  发行债券的手续费          → 计入**应付债券——利息调整**

所以：
  ❌ 不是计入长投成本
  ✅ 应计入管理费用

分录：
  借：管理费用              （审计费、律师费等）
      贷：银行存款

💡 为什么中介费不进长投成本？
  因为合并发生的交易费用是"购买服务的支出"，
  不是购买被合并方股权的对价的一部分，
  所以直接费用化。
```

---

### 第11题

> 非同一控制下企业合并中，购买日合并成本大于被购买方可辨认净资产公允价值份额的差额，应确认为商誉，并逐年摊销。

**我的答案：** ____　　**正确答案：** ❌ 错

**解析：**

```
前半句 ✅：正商誉 = 合并成本 - 可辨认净资产公允价值×持股比例
后半句 ❌：商誉**不摊销**，每年做**减值测试**！

商誉的处理：
  确认后：列为资产
  后续：❌ 不摊销
         ✅ 每年年末进行减值测试
         如果减值 → 借：资产减值损失  贷：商誉减值准备
         减值一旦确认 → 不得转回！

⚠️ 易混：
  无形资产 → 摊销（使用寿命有限）或减值测试（使用寿命不确定）
  商誉     → 不摊销，只做减值测试

分录（购买日确认商誉）：
  借：股本                  2,500万
      资本公积              6,845万
      盈余公积                500万
      未分配利润            1,005万
      商誉                  1,155万  ← 差额
      贷：长期股权投资            8,750万
          少数股东权益            3,255万
```

### 第12题

> 同一控制下企业合并中，合并方在合并日编制合并财务报表时，应当恢复被合并方在合并前实现的留存收益中归属于合并方的部分。

**我的答案：** ____　　**正确答案：** ✅ 对

**解析：**

```
这是同控合并**最容易被遗忘**的步骤！⚠️

逻辑：
  同控合并 = 集团内部重组
  被合并方在合并前赚的留存收益（盈余公积+未分配利润），
  站在最终控制方角度看，这些早就存在了。
  合并报表中如果不恢复，留存收益就会"消失"，
  导致合并报表留存收益偏低。

分录：
  借：资本公积——股本溢价    3,000万  ← 从资本公积挪回来
      贷：盈余公积                 1,000万
          未分配利润               2,000万

限制：只能恢复资本溢价余额足够覆盖的部分，
      不够的话有多少恢复多少。

💡 为什么冲资本公积？
  因为同控合并中，长投按账面价值入账，
  差额走的也是资本公积。恢复留存收益时，
  相当于把之前"多记"的资本公积转回去。
```

### 第13题

> 非同一控制下企业合并，购买日被购买方存货的评估增值部分，在该存货对外销售时，应在合并报表中调减营业成本。

**我的答案：** ____　　**正确答案：** ❌ 错

**解析：**

```
逻辑链（你的错题！）：

购买日：存货账面255万 → 公允450万（增值195万）
  借：存货              195万
      贷：资本公积           195万

当期对外销售：
  子公司个别报表：按账面255万结转营业成本
  合并层面：存货在合并报表中是450万
  
  → 营业成本应从255万"调增"到450万
  → 调增195万，不是调减！

正确分录：
  借：营业成本            195万  ← 调增！
      贷：存货                 195万

💡 本质：非同控下，子公司资产负债在合并层面是
  按公允价值持续计算的。评估增值的存货对外销售时，
  合并层面要多确认一笔营业成本。
```

### 第14题

> 企业合并中，合并方发行股票支付的佣金手续费，应计入管理费用。

**我的答案：** ____　　**正确答案：** ❌ 错

**解析：**

```
再次确认三类费用的处理：

    费用类型             处理方式                    原因
  ─────────────────────────────────────────────────────
  中介费用              → 管理费用 ✅              购买服务的支出
  发行股票佣金手续费      → 冲减资本公积——股本溢价 ✅    股票发行费用，冲溢价
  发行债券手续费          → 计入应付债券——利息调整 ✅   债券发行费用，调整面值

分录：
  借：资本公积——股本溢价    （佣金手续费）
      贷：银行存款

如果资本溢价不够冲：
  依次冲减盈余公积、未分配利润

💡 速记：
  "融资费用冲权益/调负债，交易费用进损益"
```

### 第15题

> 非同一控制下企业合并中，或有对价（以现金补偿）在购买日后12个月内的公允价值变动，应调整合并成本。

**我的答案：** ____　　**正确答案：** ❌ 错

**解析：**

```
或有对价的后续处理是个重要考点！

规则：
  购买日后**12个月内**出现新信息，表明购买日已存在的情况发生变化
  → 调整合并成本 ✅（等于是购买日没估准，补调）

  购买日后**超过12个月**的新情况，或购买日已确认的后续变动
  → 计入**公允价值变动损益** ❌（不是调整合并成本）

  以现金补偿的或有对价 → 分类为**交易性金融负债/资产**
  后续公允价值变动 → 计入**公允价值变动损益**

例（课件案例题8③）：
  购买日预计净利润1,200万，或有应付2,400万：
  借：长期股权投资          7,400万
      贷：银行存款                5,000万
          交易性金融负债            2,400万

  实际净利润800万，应支付1,600万：
  借：交易性金融负债          2,400万
      贷：银行存款                1,600万
          投资收益                  800万  ← ⚠️ 差额进损益！

注意这个例子里是超过12个月的情况，所以差额进损益。

💡 考试口诀：
  "12月内新信息→调成本，超12月新情况→进损益"
```

## 四、合并财务报表（5题）

---

### 第16题

> 母公司对子公司的长期股权投资与子公司所有者权益项目进行抵销时，差额应确认为商誉。

**我的答案：** ____　　**正确答案：** ❌ 错

**解析：**

```
抵销分录的逻辑：

抵销分录：
  借：股本                      （子公司账面）
      资本公积                   （子公司账面+公允调增）
      盈余公积                   （子公司账面）
      未分配利润                 （子公司账面）
      商誉                       ← 合并成本 - 可辨认净资产公允×持股比例
      贷：长期股权投资               （母公司账面）
          少数股东权益               （可辨认净资产公允×少数股权比例）

⚠️ 关键点：
  ① 不叫"差额确认为商誉"——商誉是在**购买日单独计算**的！
  ② 商誉 = 合并成本 - 购买日可辨认净资产公允价值×持股比例
  ③ 抵销分录中，商誉是把计算好的商誉"放进来"
  ④ 商誉只在**非同控**下产生，同控下没有商誉

💡 可以理解为：
  你先算好商誉是多少（像算好一个数）
  然后在抵销分录里把这个数填进去（差额=商誉）
  不是"差额自动变成商誉"！
```

### 第17题

> 合并利润表中的"归属于少数股东的损益"应按子公司净利润乘以少数股东持股比例计算，不受内部交易影响。

**我的答案：** ____　　**正确答案：** ❌ 错

**解析：**

```
⚠️ 内部交易会影响少数股东损益！

规则：
  少数股东损益 = 子公司**调整后**净利润 × 少数股东持股比例
  
  调整：要扣除内部交易中**逆流交易**（子公司卖给母公司）
        未实现利润中归属于少数股东的部分

例：
  子公司净利润          = 1,000万
  子公司向母公司销售商品，未实现利润 = 100万（逆流交易！）
  
  调整后净利润          = 1,000 - 100 = 900万
  少数股东持股比例      = 30%
  少数股东损益          = 900 × 30% = 270万  ✅（不是1,000×30%）

💡 为什么逆流交易要调整？
  未实现利润在子公司账上，子公司净利润虚高100万→要砍掉
  ⚠️ 顺流交易（母公司卖给子公司）的未实现利润**全归母公司承担**，
  不影响少数股东损益！
```

### 第18题

内部存货交易的抵销：

① 抵销内部销售

  借：营业收入              1,000万

## 六、非货币性资产交换（4题）

---

### 第25题

> 非货币性资产交换中，支付的补价占换入资产公允价值（或换出资产公允价值与支付补价之和）的比例不超过25%的，视为非货币性资产交换。

**我的答案：** ____　　**正确答案：** ✅ 对

**解析：**

```
比例计算公式（你之前错过的点！）：

收到补价方：
  补价比例 = 收到的补价 ÷ 换出资产公允价值

支付补价方：
  补价比例 = 支付的补价 ÷（换出资产公允价值 + 支付的补价）

判定：
  ≤ 25% → 属于非货币性资产交换 ✅
  > 25% → 按正常买卖处理 ❌

⚠️ **必用不含税价**计算！（考纲特别强调！）

例：换出公允100万（增值税13%），换入公允120万（增值税13%）
  含税补价 = 120×1.13 - 100×1.13 = 135.6 - 113 = 22.6万
  不含税补价 = 120 - 100 = 20万

  ❌ 22.6÷100 = 22.6% → 但这是错的！用含税价算！
  ✅ 20÷100 = 20% → 正确！不含税价！
  ✅ 20÷(100+20) = 16.67% → 支付方正确算法
```

### 第26题

> 具有商业实质的非货币性资产交换中，换出资产为固定资产的，其公允价值与账面价值的差额应计入营业外收入或营业外支出。

**我的答案：** ____　　**正确答案：** ❌ 错

**解析：**

```
⚠️ 科目混用是高频错误！

          换出资产类型               损益科目
  ──────────────────────────────────────────
  固定资产、无形资产、在建工程    → 资产处置损益 ✅
  存货                          → 主营业务收入（并结转成本）
  长期股权投资、交易性金融资产    → 投资收益 ✅
  投资性房地产                   → 其他业务收入（公允价值模式）
                               → 资产处置损益（成本模式→账面转公允）

固定资产换出的完整分录：
  ① 转入清理
  借：固定资产清理
      累计折旧
      贷：固定资产
  
  ② 交换
  借：换入资产
      贷：固定资产清理              （换出资产**公允**价值）
          银行存款等
  
  ③ 确认损益
  借：固定资产清理                  （公允-账面）
      贷：**资产处置损益** ✅          （不是营业外收入/支出！）

💡 为什么现在用"资产处置损益"？
  2017年修订后，固定资产和无形资产的处置损益从"营业外收支"
  改为"资产处置损益"，因为它属于**日常经营活动中的资产处置**，
  不是偶发事件。
  ⚠️ 只有报废、毁损等非正常处置才用营业外收支。
```

## 七、债务重组（3题）

---

### 第29题

> 债务重组中，债权人应将受让资产的入账价值按该资产的公允价值计量，差额计入投资收益。

**我的答案：** ____　　**正确答案：** ❌ 错

**解析：**

```
⚠️ 这是债务重组最经典的陷阱！

债权人换入资产的入账规则：
  ❌ 不是按"换入资产的公允价值"入账
  ✅ 按**放弃债权的公允价值**入账！

公式：
  换入资产入账价值 = 放弃债权的公允价值 - 增值税进项税额 + 相关税费

损益确认：
  放弃债权的公允价值 - 应收账款账面价值 = **投资收益**

例（你的笔记）：放弃债权公允95万，应收账款账面95万（原值100-坏账5）
    收到固定资产公允90万，增值税11.7万，支付运费4万

    固定资产入账价值 = 放弃债权公允95 - 进项税11.7 = 83.3万 ？

    完整分录：
    借：固定资产              91万  ← 95-4（运费部分？）
        应交税费——进项税      11.7万
        投资收益               1万  ← 差额损失
        坏账准备               5万
        贷：应收账款               100万
            银行存款               4万
            应交税费——增值税（略）

📌 关键是：**放弃债权的公允价值**是计量的锚点！

💡 与非货币性资产交换的区别：
  非货币性交换 → 按**换出资产公允**计价
  债务重组     → 按**放弃债权的公允**计价
```

### 第30题

> 债务人以资产清偿债务进行债务重组的，债务人应将所清偿债务的账面价值与转让资产账面价值之间的差额计入其他综合收益。

**我的答案：** ____　　**正确答案：** ❌ 错

**解析：**

```
⚠️ 两个错误：科目错了 + 用的是账面价值

正确做法：
  差额 = 债务的账面价值 - 转让资产的账面价值
  → 计入**其他收益——债务重组收益** ✅（不是其他综合收益！）

例（你的笔记）：应付账款95万，用固定资产清偿
    固定资产清理价值88万

  借：应付账款              95万
      贷：固定资产清理            88万
          其他收益——债务重组收益     7万  ← 差额！

其他收益——债务重组收益是什么科目？
  利润表中的**营业收入类**科目
  专门用于核算债务重组中债务人的利得

债务人不同重组方式的损益科目：

    重组方式                损益科目
  ──────────────────────────────────
  以资产清偿债务          → 其他收益——债务重组收益 ✅
  将债务转为权益工具      → 其他收益——债务重组收益 ✅
  修改其他条款            → 其他收益——债务重组收益 ✅
  组合方式                → 其他收益——债务重组收益 ✅

💡 速记：
  "债务人永远用'其他收益——债务重组收益'"
  "债权人永远用'投资收益'"
```
