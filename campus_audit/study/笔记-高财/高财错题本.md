### ❌ 错题1：同控下中介费用的处理

> **来源**：学习通作业2 · 第7题
> 
> 同一控制下企业合并中，合并方发生的中介费用应计入管理费用。
> 
> **我的答案**：❌ 错　　**正确答案**：✅ 对

**考点**：企业合并交易费用的处理

**正确结论**：

|        费用类型         |       同控合并       |      非同控合并       |
| :-----------------: | :--------------: | :--------------: |
| **中介费用**（审计、法律、评估费） |    **管理费用** ✅    |    **管理费用** ✅    |
|     发行股票的佣金手续费      | 冲减**资本公积——股本溢价** | 冲减**资本公积——股本溢价** |
|      发行债券的手续费       | 计入**应付债券——利息调整** | 计入**应付债券——利息调整** |

**📝 错因分析**：我搞混了中介费用和发行费用的去向。**中介费用无论同控还是非同控，都计入管理费用**，不会计入长投成本。

> 详见 [[study/笔记-高财/高级财务会计-企业合并#^选择题高频易错点汇总|企业合并易错点⑨]]

### ❌ 错题2：非同控购买日存货评估增值的后续处理

> **来源**：学习通作业2 · 第12题
> 
> 非同一控制下企业合并中，购买日存货评估增值，若该存货当期对外销售，合并报表中应调增营业成本。
> 
> **我的答案**：❌ 错　　**正确答案**：✅ 对

**考点**：非同控下公允价值调整在合并报表中的后续影响

**推理过程**：
```
购买日：存货账面255万 → 公允450万（增值195万）
        借：存货              195万
            贷：资本公积           195万

当期对外销售：存货从账面价值结转成本，但合并层面该存货成本是公允价值！
        借：营业成本            195万  ← 调增！
            贷：存货                 195万
```

**📝 错因分析**：我忘了非同控下子公司资产负债在合并层面是按**公允价值**持续计算的。评估增值的存货对外销售时，合并层面要多确认一笔营业成本。

> 详见 [[第27章 合并财务报表（全）|合并报表章节]]

### ❌ 错题3：外币应付账款汇兑差额的去向

> **来源**：学习通作业2 · 第22题
> 
> 甲公司记账本位币为人民币，外币应付账款在6月份因汇率变动产生的差额应计入其他综合收益。
> 
> **我的答案**：✅ 对　　**正确答案**：❌ 错

**考点**：外币货币性项目汇兑差额的归属

```
外币应付账款 → 货币性项目 → 期末按即期汇率调整
差额 → 计入「财务费用——汇兑差额」（不是其他综合收益！）
```

**📝 错因分析**：我把**外币报表折算差额**（记其他综合收益）和**外币交易汇兑差额**（记财务费用）搞混了。

|            场景            |    差额去向    |
| :----------------------: | :--------: |
| 外币**交易**（应收账款、应付账款等）期末调整 |  **财务费用**  |
|  外币**报表折算**（境外子公司报表折算）   | **其他综合收益** |
|    外币**其他权益工具投资**汇率变动    | **其他综合收益** |
|     外币**其他债权投资**汇兑差额     |  **财务费用**  |

> 详见 [[22 第二十二章 外币折算(PDF笔记版)|外币折算章节]]

### ❌ 错题4：使用寿命不确定的无形资产与暂时性差异

> **来源**：学习通作业2 · 第23题
> 
> 使用寿命不确定的无形资产，因不摊销，账面价值与计税基础不会产生暂时性差异。
> 
> **我的答案**：✅ 对　　**正确答案**：❌ 错

**考点**：资产计税基础与账面价值的差异

**推理过程**：
```
使用寿命不确定的无形资产 → 不摊销，但每年要做减值测试

账面价值 = 原值 - 减值准备（如果减值了）
计税基础 = 原值 - 税法允许的摊销（税法不管你是否确定使用寿命，都让摊销！）

→ 即使不摊销，如果计提了减值准备，账面价值≠计税基础
→ 会产生暂时性差异
```

**📝 错因分析**：我想当然地以为"不摊销就没有差异"，但忽略了**减值测试**和**税法摊销**的存在。

### ❌ 错题5：交易性金融资产与暂时性差异

> **来源**：学习通作业2 · 第24题
> 
> 已确认公允价值变动损益的交易性金融资产，账面价值与计税基础可能形成暂时性差异。
> 
> **我的答案**：❌ 错　　**正确答案**：✅ 对

**考点**：以公允价值计量的金融资产的计税基础

```
交易性金融资产：
  账面价值 = 公允价值
  计税基础 = 初始取得成本（税法不承认公允价值的变动）

账面价值 ≠ 计税基础 → 形成暂时性差异！
账面价值 > 计税基础 → 应纳税暂时性差异 → 递延所得税负债
```

**📝 错因分析**：我误以为公允价值变动损益已经影响会计利润了，所以不会有差异。但税法**不承认**未实现的公允价值变动，所以账面和计税基础确实不同。

### ❌ 错题6：持有至到期投资换入长投是否属于非货币性资产交换

> **来源**：学习通作业2 · 第27题
> 
> 持有至到期投资（债券）换入长期股权投资，属于非货币性资产交换。
> 
> **我的答案**：✅ 对　　**正确答案**：❌ 错

**考点**：货币性资产 vs 非货币性资产的判断

|       资产        |    性质     | 原因                |
| :-------------: | :-------: | :---------------- |
| **持有至到期投资**（债券） | **货币性资产** | 金额固定或可确定，到期收回固定金额 |
|   **长期股权投资**    |  非货币性资产   | 金额不固定，取决于被投资方经营状况 |

```
持有至到期投资（货币性）→ 长期股权投资（非货币性）
→ 涉及货币性资产的交换 → 不是非货币性资产交换！
```

**📝 错因分析**：我把"持有至到期投资"误当成非货币性资产了。**债券类投资都是货币性资产**（因为未来收回的金额是确定的）。

> ⚠️ **速记**：收回的钱是固定的 → 货币性资产
> 
> 货币性资产：库存现金、银行存款、**应收账款、应收票据、持有至到期投资、债权投资**
> 
> 非货币性资产：存货、固定资产、无形资产、**长期股权投资、交易性金融资产**、投资性房地产、**预付账款**

### ❌ 错题8：非同控下商誉的计算

> **来源**：学习通作业2 · 第30题
> 
> 非同一控制下企业合并，购买日应确认的商誉等于合并成本减去享有被购买方可辨认净资产账面价值的份额。
> 
> **我的答案**：✅ 对　　**正确答案**：❌ 错

**考点**：商誉的计算公式

```
❌ 错误公式：商誉 = 合并成本 - 可辨认净资产账面价值 × 持股比例
✅ 正确公式：商誉 = 合并成本 - 可辨认净资产公允价值 × 持股比例
```

**📝 错因分析**：**非同控**下子公司资产负债是按**公允价值**计量的，所以要用可辨认净资产的**公允价值**份额，不是账面价值！只有**同控**才用账面价值。

> 详见 [[study/笔记-高财/高级财务会计-企业合并#商誉的计算|企业合并商誉计算]]

### ❌ 错题9：超过标准的广告费是否属于永久性差异

> **来源**：学习通高财作业 · 第2题
> 
> 企业发生的超过税法扣除标准的广告费支出，属于永久性差异。
> 
> **我的答案**：✅ 对　　**正确答案**：❌ 错

**考点**：永久性差异 vs 暂时性差异的区分

```
永久性差异：当期发生，税法永远不让扣（如罚款、赞助支出）
暂时性差异：当期不让扣，但以后期间可以扣（如广告费、职工教育经费）

超过税法扣除标准的广告费 → 可以结转以后年度扣除
→ 属于可抵扣暂时性差异，不是永久性差异！
```

**📝 错因分析**：我误以为"超过标准的部分"就是永久性差异。但广告费超过部分可以**结转以后年度扣除**，所以是可抵扣暂时性差异。

**常见项目归类**：

| 永久性差异（永远不能扣） |        暂时性差异（以后可以扣）        |
| :----------: | :------------------------: |
|     行政罚款     |    **广告费超过部分**（结转以后年度）     |
|  国债利息收入（免税）  |     职工教育经费超过部分（结转以后年度）     |
|    税收滞纳金     | 业务招待费超过部分（**不能结转，是永久性**⚠️） |
|     赞助支出     |     计提的资产减值准备（实际发生时扣除）     |

> ⚠️ 注意：业务招待费超过部分虽然也是超过标准，但**不能结转以后年度**，属于**永久性差异**！

### ❌ 错题10：无商业实质的非货币性交换是否确认损益

> **来源**：学习通高财作业 · 第17题
> 
> 不具有商业实质的非货币性资产交换，无论是否支付补价，均不确认损益。
> 
> **我的答案**：❌ 错　　**正确答案**：✅ 对

**考点**：非货币性资产交换两种计价模式的选择

```
不具有商业实质 → 账面价值模式 → 不确认损益（差额不进利润表）
具有商业实质 → 公允价值模式 → 确认损益（差额进资产处置损益/主营业务收入等）
```

**📝 错因分析**：我可能觉得"支付了补价就应该确认损益"。但**无商业实质的本质是：交换前后企业的经济状况没有实质性变化**，所以不允许确认损益，防止利润操纵。

> 详见 [[study/笔记-高财/整理-非货币性资产交换#两种计价模式|非货币性资产交换]]
