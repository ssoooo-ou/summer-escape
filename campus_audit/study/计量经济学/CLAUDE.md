### 第一章 导论 `[选择/判断为主]`

**掌握内容：**

1. **计量经济学研究步骤** — 熟记完整流程（模型设定 → 估计 → 检验 → 预测/政策分析）。
2. **四大数据类型** — 重点区分：
   - 时间序列数据（同一对象、不同时间）
   - 截面数据（同一时点、不同对象）
   - 面板数据（时间+截面混合）
   - 虚拟变量数据
3. **变量分类**：
   - 按因果：解释变量（自变量/X）vs 被解释变量（因变量/Y）
   - 按属性：内生变量（模型内决定）vs 外生变量（模型外给定）
4. **三类可转换模型**（重点3个）：
   - **双对数模型** lnY = β₀ + β₁lnX + ε → β₁ 代表**弹性**（X 变化 1%，Y 变化 β₁%）
   - **半对数模型** lnY = β₀ + β₁X + ε → β₁ 代表**增长率**
   - **倒数模型** Y = β₀ + β₁/X + ε

> 📌 教学提示：这一章只出选择判断，让学生能选对就行，不需要深入推导。

### 第二章 简单线性回归模型 `[计算必考]` `[简答高频]` ⭐⭐⭐

这是整门课**最重要的章节**，考试的计算题和证明题主要出在这里。

#### 2.1 核心基础概念 `[选择/判断]`

- **三大核心定义**：
  - 总体回归线（PRF）：E(Y|X) = β₀ + β₁X，描述的是总体的真实关系
  - 样本回归线（SRF）：Ŷ = β̂₀ + β̂₁X，是用样本数据估计出来的
  - 随机误差项 εᵢ = Yᵢ - E(Y|Xᵢ)：包含了所有未被 X 解释的因素

- **两组区分**：
  - 总体回归函数 vs 样本回归函数：联系（SRF 是对 PRF 的估计）、区别（一个是理论真实值、一个是用样本算出来的估计值）
  - 相关分析 vs 回归分析：相关分析度量关联程度（对称）、回归分析研究因果方向（不对称）

- **总体参数 vs 样本统计量**：总体参数（β₀、β₁）是唯一确定的；样本统计量（β̂₀、β̂₁）是随机的、服从正态分布

#### 2.2 五大古典基本假定 `[简答必背]`

> 📌 **必须一字不差地背下来，简答题高频！**

| 假定 | 内容 | 数学表达 |
|------|------|---------|
| 假定1 | 线性假定：Y 与 X 之间存在线性关系 | Yᵢ = β₀ + β₁Xᵢ + εᵢ |
| 假定2 | X 是确定性的（非随机） | 或在重复抽样中 X 值固定 |
| 假定3 | 零均值：随机误差项 εᵢ 的期望为零 | E(εᵢ) = 0 |
| 假定4 | 同方差：所有 εᵢ 的方差相同 | Var(εᵢ) = σ²（常数） |
| 假定5 | 无自相关：不同观测的 εᵢ 不相关 | Cov(εᵢ, εⱼ) = 0（i≠j） |

> 📌 教学提示：要求学生先背全五条，再解释每条违反的后果。

#### 2.3 普通最小二乘法（OLS）推导 `[计算必考]` `[证明必考]`

**核心原理**：残差平方和（RSS）最小化。

**推导过程（必须能默写）**：

```
目标函数：min Σeᵢ² = Σ(Yᵢ - β̂₀ - β̂₁Xᵢ)²

Step 1：对 β̂₀ 求偏导，令其 = 0
    ∂/∂β̂₀ Σ(Yᵢ - β̂₀ - β̂₁Xᵢ)² = -2Σ(Yᵢ - β̂₀ - β̂₁Xᵢ) = 0
    → ΣYᵢ = nβ̂₀ + β̂₁ΣXᵢ …… (1)  ← 正规方程一

Step 2：对 β̂₁ 求偏导，令其 = 0
    ∂/∂β̂₁ Σ(Yᵢ - β̂₀ - β̂₁Xᵢ)² = -2Σ(Yᵢ - β̂₀ - β̂₁Xᵢ)Xᵢ = 0
    → ΣXᵢYᵢ = β̂₀ΣXᵢ + β̂₁ΣXᵢ² …… (2)  ← 正规方程二

Step 3：联立解出
    β̂₁ = Σ(Xᵢ-X̄)(Yᵢ-Ȳ) / Σ(Xᵢ-X̄)²  =  Σxᵢyᵢ / Σxᵢ²
    其中 xᵢ = Xᵢ-X̄（离差形式），yᵢ = Yᵢ-Ȳ

    β̂₀ = Ȳ - β̂₁X̄
```

> 📌 教学提示：要求学生不仅会背公式，还要能手写出完整推导过程（证明题可能直接考）。

#### 2.4 OLS 回归线的 5 个性质 `[简答/选择]`

1. 回归线经过样本均值点 (X̄, Ȳ)
2. Ȳ̂ 的均值 = Ȳ（Y 的实际均值 = 预测均值）
3. 残差均值为零：Σeᵢ = 0
4. 残差与 X 不相关：ΣeᵢXᵢ = 0
5. 残差与预测值不相关：ΣeᵢŶᵢ = 0

#### 2.5 必考公式清单 `[计算必考]`

**① 随机扰动项方差无偏估计**：
$$\hat{\sigma}^2 = \frac{SSE}{n-2} = \frac{\sum e_i^2}{n-2}$$

- SSE = Σeᵢ² = 残差平方和
- n-2 = 自由度（为什么是 n-2？因为估计了2个参数 β̂₀ 和 β̂₁，损失了2个自由度）

**② 回归系数方差与标准差**：
$$Var(\hat{\beta}_1) = \frac{\sigma^2}{\sum (X_i-\bar{X})^2} = \frac{\sigma^2}{\sum x_i^2}$$
$$Var(\hat{\beta}_0) = \sigma^2\left(\frac{1}{n} + \frac{\bar{X}^2}{\sum (X_i-\bar{X})^2}\right)$$
$$SE(\hat{\beta}_j) = \sqrt{Var(\hat{\beta}_j)}$$

**标准差的作用**：用于 t 检验 —— t = β̂ⱼ / SE(β̂ⱼ)

> 📌 考试中绝对会用到：给你 SSE、n、Σxᵢ²，让你一步步算 → σ̂² → Var(β̂₁) → SE(β̂₁) → t统计量。

#### 2.6 三大假设检验 `[计算必考]` `[简答高频]`

| 检验 | 检验对象 | 统计量 | 原假设 H₀ | 拒绝域 |
|------|---------|--------|----------|--------|
| **拟合优度 R²** | 模型整体解释力 | R² = ESS/TSS = 1 - RSS/TSS | — | 越接近1越好 |
| **t 检验** | 单个回归系数 | t = β̂₁/SE(β̂₁) ~ t(n-2) | H₀: β₁ = 0 | \|t\| > t_{α/2}(n-2) |
| **F 检验** | 所有系数联合显著 | F = (ESS/1)/(RSS/(n-2)) = MSR/MSE | H₀: β₁ = 0 | F > F_α(1, n-2) |

**R² 的公式推导（必须理解）**：

TSS = Σ(Yᵢ-Ȳ)²（总离差平方和）
ESS = Σ(Ŷᵢ-Ȳ)²（回归平方和，X 能解释的部分）
RSS = Σeᵢ² = Σ(Yᵢ-Ŷᵢ)²（残差平方和，X 不能解释的部分）

TSS = ESS + RSS（方差分解，利用 ΣeᵢŶᵢ = 0）

→ R² = ESS/TSS = 1 - RSS/TSS

**R² 的经济含义**：Y 的总变异中，能被 X 的线性变动解释的比例。取值 0~1。

#### 2.7 回归模型预测 `[计算]`

- **点预测**：X = X₀ 时，Ŷ₀ = β̂₀ + β̂₁X₀  直接代入
- **Y 平均值的区间预测**：E(Y|X₀) 的置信区间（课本公式 2.76）——较窄
- **Y 个体值的区间预测**：Y₀ 的预测区间（课本公式 2.88）——较宽

> 📌 个体值预测区间 > 平均值置信区间，因为个体值多了 ε 的不确定性。

### 第三章 多元线性回归模型 `[计算必考]` `[简答高频]` ⭐⭐⭐

#### 3.1 模型形式

一般形式：Yᵢ = β₁ + β₂X₂ᵢ + β₃X₃ᵢ + ... + βₖXₖᵢ + uᵢ

**偏回归系数** βⱼ 的含义：**控制其他解释变量不变**的条件下，第 j 个解释变量每变动 1 个单位，Y 平均变动 βⱼ 个单位。

矩阵形式：**Y = Xβ + u**

#### 3.2 多元古典假定（5 条）

与第二章基本一致，新增一条关键假定：

| 新增假定 | 内容 |
|---------|------|
| **无多重共线性** | 解释变量之间不存在精确的线性关系，即 X 矩阵满秩 rank(X) = k |

#### 3.3 OLS 估计（矩阵形式）`[计算/证明]`

$$\hat{\boldsymbol{\beta}} = (\boldsymbol{X}'\boldsymbol{X})^{-1}\boldsymbol{X}'\boldsymbol{Y}$$

**推导思路**：
1. 目标：min Σeᵢ² = e'e = (Y-Xβ̂)'(Y-Xβ̂)
2. 对 β̂ 向量求导，令其 = 0
3. 得：X'Xβ̂ = X'Y
4. 若 X'X 可逆，则 β̂ = (X'X)⁻¹X'Y

> 📌 二元回归的复杂代数展开公式考频极低（老师说的），但矩阵形式的表达式需要记住。

#### 3.4 调整后的可决系数 R̄² `[简答/计算]`

$$\bar{R}^2 = 1 - \frac{RSS/(n-k)}{TSS/(n-1)} = 1 - (1-R^2)\frac{n-1}{n-k}$$

**为什么要调整？** 普通 R² 在增加解释变量时只增不减（即使加了无关变量），R̄² 引入了自由度惩罚（k 越大，惩罚越大），只有新增变量对 Y 解释力足够大时 R̄² 才会上升。

**R² 与 R̄² 的关系**：R̄² ≤ R²；R̄² 可能为负数（当模型拟合极差时）。

#### 3.5 F 统计量与 R² 的关联 `[必记公式]`

$$F = \frac{R^2/(k-1)}{(1-R^2)/(n-k)}$$

> 📌 这是课本公式 3.48，考试可能直接考。它揭示了 F 检验和 R² 的内在联系：R² 越大 → F 统计量越大 → 越可能拒绝 H₀。

#### 3.6 检验和预测

t 检验、F 检验、预测的逻辑与第二章一致，不需要另外记忆新的逻辑框架。

### 第四章 多重共线性 `[简答高频]` `[选择/判断]`

#### 4.1 基本概念
多重共线性是指**解释变量之间存在近似线性关系**（完全共线性是精确线性关系）。

#### 4.2 五大影响（后果）（背下来！）`[简答必考]`

1. 参数估计值方差增大（OLS 估计量不再有效）
2. 置信区间变宽
3. t 检验不显著（t 值变小，容易错误接受 H₀）
4. R² 可能仍然很高（但 t 检验多不显著——经典矛盾信号）
5. 参数估计值对样本变化非常敏感（增减几个样本，系数大幅波动）

#### 4.3 方差膨胀因子（VIF）`[重点]`

$$VIF_j = \frac{1}{1 - R_j^2}$$

其中 R²ⱼ 是以 Xⱼ 为被解释变量、其余 X 为解释变量做回归的可决系数。

- VIF > 10（或 R²ⱼ > 0.9）→ 存在严重多重共线性
- VIF = 1 → Xⱼ 与其他解释变量完全不相关

#### 4.4 检验方法 `[简答]`
- 相关系数矩阵法（两两变量间相关系数 > 0.8 则可疑）
- 辅助回归法（对每个 X 做辅助回归，看 R²ⱼ）
- VIF 法
- 不显著系数法（F 检验显著但多数 t 检验不显著 → 多重共线性的经典信号）

#### 4.5 修正方法 `[简答]`
- 删除变量（删除引起共线性的变量）
- 增加样本量
- 变量变换（如差分变换、取对数）
- 岭回归（有偏估计，牺牲无偏性换取方差降低）

### 第五章 异方差性 `[简答高频]` `[计算可能]` ⭐⭐

#### 5.1 基本概念

异方差 = 随机误差项的**方差不恒定**：Var(εᵢ) = σᵢ² ≠ 常数。违背了古典假定 4。

**产生原因**（三点）：数据收集技术改进（σ² 随时间变小）、模型设定偏误、异常值。

#### 5.2 对模型的影响 `[简答]`

| 影响 | 说明 |
|------|------|
| OLS 仍无偏 | ✅ 异方差不影响无偏性 |
| OLS 不再有效（最小方差性丧失） | ❌ 参数估计量的方差不是最小的 |
| 标准误/方差的估计有偏 | t 检验、F 检验失效 |
| 预测区间变宽 | 预测精度下降 |

> ⚠️ 对比记忆：异方差不影响无偏性，但影响**有效性**！

#### 5.3 检验方法（四种，本部分核心）`[简答必考]`

> 📌 老师特别强调：「怀特检验和 ARCH 检验的区别是什么？这类简答题今年很有可能会考到。一定要把这四种检验的统计量区别搞清楚。」

| 检验方法 | 辅助回归 | 检验统计量 | 适用场景 |
|---------|---------|-----------|---------|
| **GQ 检验**（Goldfeld-Quandt） | 将样本分为两组，分别回归 | F = RSS₂/RSS₁ ~ F | 异方差随 X 单调变化 |
| **怀特（White）检验** | eᵢ² 对所有 X、X²、交叉项回归 | n×R² ~ χ² | 任何形式的异方差 |
| **ARCH 检验** | eₜ² 对 eₜ₋₁², eₜ₋₂², ... 回归 | n×R² ~ χ² | **时间序列**的异方差（自回归条件异方差） |
| **BPG 检验**（Breusch-Pagan-Godfrey） | eᵢ² 对 X 回归 | n×R² ~ χ² | 线性形式的异方差 |

**White 检验 vs ARCH 检验的区别** `[简答高频]`：
- White 检验：用截面数据，辅助回归的变量是所有解释变量及其平方、交叉项
- ARCH 检验：用**时间序列**数据，辅助回归的变量是**残差平方的滞后项**（eₜ₋₁², eₜ₋₂², ...）

#### 5.4 修正方法 `[简答/计算]`

**① 模型变换法（重点）**：已知异方差形式 σᵢ² = σ²f(Xᵢ) 时，等式两边同除以 √f(Xᵢ) 做 WLS。
- 例：若 Var(εᵢ) = σ²Xᵢ²，则原方程除以 Xᵢ → Yᵢ/Xᵢ = β₀/Xᵢ + β₁ + εᵢ/Xᵢ

**② 加权最小二乘法（WLS）**：给予方差较小的观测更大权重。

### 第六章 自相关 `[简答高频]` `[计算可能]` ⭐⭐

#### 6.1 基本概念

自相关 = 不同观测点的随机误差项之间存在相关关系：Cov(εᵢ, εⱼ) ≠ 0（i≠j）。违背古典假定 5。

**一阶自相关（AR(1)）**：εₜ = ρεₜ₋₁ + vₜ，其中 |ρ| < 1，vₜ 满足古典假定。

**产生原因**（三点）：经济惯性（经济变量的持续性）、模型设定偏误（遗漏重要变量）、数据处理（如内插、平滑）。

#### 6.2 对模型的影响 `[简答]`

| 影响 | 说明 |
|------|------|
| OLS 仍无偏 | ✅（与异方差一样） |
| OLS 不再有效 | ❌（最小方差性丧失） |
| 方差估计有偏（通常低估） | t 检验被夸大、容易错误拒绝 H₀ |
| 预测精度下降 | 预测区间不可靠 |

#### 6.3 DW 检验 `[计算必考]` ⭐⭐⭐

$$DW = \frac{\sum_{t=2}^{n} (e_t - e_{t-1})^2}{\sum_{t=1}^{n} e_t^2}$$

近似关系：DW ≈ 2(1 - ρ̂)，其中 ρ̂ 是一阶自相关系数估计值。

**DW 判定规则**（给定显著性水平 α，查 dL 和 dU）：

| DW 值范围 | 结论 |
|----------|------|
| 0 < DW < dL | 存在正自相关 |
| dL ≤ DW ≤ dU | **无结论区**（不能确定） |
| dU < DW < 4-dU | 无自相关 ✅ |
| 4-dU ≤ DW ≤ 4-dL | **无结论区**（不能确定） |
| 4-dL < DW < 4 | 存在负自相关 |

> 📌 这是计算题的经典考点：给你 DW 统计量和 dL/dU，让你判断是否自相关。

**DW 检验的局限（限制条件）** `[简答]`：
1. 只能检验一阶自相关
2. 回归方程必须包含截距项
3. 解释变量必须是确定性的（不含 Y 的滞后项）
4. 存在两个无结论区
5. 样本量不能太小

#### 6.4 LM 检验（拉格朗日乘数检验）`[简答]`

- 辅助回归：eₜ 对 X 和 eₜ₋₁, eₜ₋₂, ..., eₜ₋ₚ 回归
- 检验统计量：LM = n×R² ~ χ²(p)（p 为滞后阶数）
- 优点：可以检验高阶自相关；克服了 DW 存在的局限。

#### 6.5 广义差分法（修正方法）

已知 ρ → 做广义差分：Yₜ - ρYₜ₋₁ = β₀(1-ρ) + β₁(Xₜ - ρXₜ₋₁) + vₜ

未知 ρ → 用 DW 估算 ρ̂ ≈ 1 - DW/2，再做差分（Cochrane-Orcutt 迭代法）。

### 第八章 随机解释变量 `[选择/判断]`

**核心问题**：当解释变量 X 与随机误差项 ε 相关时（Cov(X, ε) ≠ 0），OLS 估计量不再是**一致的**。

**后果**：
- 如果 X 与 ε 相关 → OLS 有偏且不一致
- 如果 X 与 ε 不相关 → OLS 仍是无偏且一致的

**解决方法**：工具变量法（IV）——找一个与 X 高度相关、但与 ε 不相关的变量 Z 作为工具变量。

### 三大违背假定的对比

| 问题 | 违背假定 | 无偏性 | 有效性 | 核心检验 | 核心修正 |
|------|---------|--------|--------|---------|---------|
| **多重共线性** | 无多重共线性（假定） | ✅ | ❌ 方差膨胀 | VIF值 | 删变量/增样本/岭回归 |

> 📌 关键记忆点：三者都不影响无偏性，但都影响有效性。

- **公式推导必须完整**：从定义出发，一步步推导，标注「为什么能这样做」。
