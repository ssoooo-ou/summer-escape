# 第四章 多重共线性笔记

## 一、基本概念

### 1. 什么是多重共线性？

**完全多重共线性**：存在不全为零的 $\lambda_2, \lambda_3, ..., \lambda_k$，使得：
$$\lambda_2 X_{2i} + \lambda_3 X_{3i} + \cdots + \lambda_k X_{ki} = 0$$
即解释变量之间存在**精确的线性关系**。（例如 $X_2 = 2X_3$）

**不完全多重共线性**：
$$\lambda_2 X_{2i} + \lambda_3 X_{3i} + \cdots + \lambda_k X_{ki} + \mu_i = 0$$
即解释变量之间存在**近似的线性关系**（$\mu_i$ 为随机误差项）。

### 2. 违背的古典假定

违背了第④条：**无多重共线性**（$X$ 矩阵列满秩 $rank(X)=k$）

### 3. 产生原因（简答考点）

| 原因          | 说明                     |
| :---------- | :--------------------- |
| ① 时间序列的共同趋势 | GDP、消费、投资都随经济周期同向变动    |
| ② 截面数据的内在关联 | 大公司规模大 → 资本、劳动、收入自然相关  |
| ③ 滞后变量的引入   | $X_t$ 和 $X_{t-1}$ 高度相关 |
| ④ 变量选择不当    | 把互相解释的变量都放进模型          |

## 二、核心概念：为什么方差会变大？⭐⭐⭐

### 关键公式

对于模型 $Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + u$：

$$Var(\hat{\beta}_1) = \frac{\sigma^2}{\sum x_{1i}^2 (1 - r_{12}^2)}$$

其中 $r_{12}^2$ 是 $X_1$ 与 $X_2$ 的相关系数平方。

### 直观理解：Frisch-Waugh-Lovell（FWL）定理

> $\hat{\beta}_1$ 衡量的是"**控制 $X_2$ 不变的情况下**，$X_1$ 的独特贡献"。

**两步法估计 $\hat{\beta}_1$：**

**第 1 步**：把 $X_1$ 中能被 $X_2$ 解释的部分去掉：
$$X_1 = \hat{\gamma}_0 + \hat{\gamma}_1 X_2 + \tilde{X}_1$$
得到残差 $\tilde{X}_1$ —— $X_1$ 的"独有部分"
**第 2 步**：用独有部分去解释 $Y$：
$$\hat{\beta}_1 = \frac{\sum \tilde{X}_{1i} Y_i}{\sum \tilde{X}_{1i}^2}, \quad Var(\hat{\beta}_1) = \frac{\sigma^2}{\sum \tilde{X}_{1i}^2}$$

**关键**：当 $X_1$ 与 $X_2$ 高度相关时，$X_1$ 几乎完全被 $X_2$ 解释 → $\tilde{X}_1$ 非常小 → 分母 $\sum \tilde{X}_{1i}^2$ 非常小 → 方差极大！

### 用解方程类比

```
正常情况（不共线）：
  X₁ +  X₂ = 10
  X₁ -  X₂ = 2
  → 唯一解：X₁=6, X₂=4  ✅

近似共线情况：
  X₁ +  X₂ = 10
  1.01X₁ + X₂ = 10.1
  → 很难精准解出 X₁ 和 X₂ ❌

完全共线情况：
  X₁ + X₂ = 10
  2X₁ + 2X₂ = 20
  → 无穷多解，无法确定 ❌❌
```

### 一个生活化例子

研究"学习时长"和"出勤次数"对成绩的影响：

| 学生 | 学习时长 | 出勤次数 | 成绩 |
|:---:|:--------:|:--------:|:---:|
| A | 1h | 1次 | 60 |
| B | 2h | 2次 | 65 |
| C | 3h | 3次 | 70 |
| D | 10h | 10次 | 95 |

> 出勤次数 = 学习时长（$r=1$）→ 完全无法区分谁的作用。
> 是学习 1h +5 分，还是出勤 1 次 +5 分？→ 方差无穷大。

### 数学推导（来自矩阵形式）

$$Var(\hat{\boldsymbol{\beta}}) = \sigma^2 (\boldsymbol{X}'\boldsymbol{X})^{-1}$$

$$\boldsymbol{X}'\boldsymbol{X} = \begin{bmatrix} \sum x_{1i}^2 & \sum x_{1i}x_{2i} \\ \sum x_{1i}x_{2i} & \sum x_{2i}^2 \end{bmatrix}$$

$$(\boldsymbol{X}'\boldsymbol{X})^{-1} = \frac{1}{\sum x_{1i}^2 \sum x_{2i}^2 - (\sum x_{1i}x_{2i})^2} \begin{bmatrix} \sum x_{2i}^2 & -\sum x_{1i}x_{2i} \\ -\sum x_{1i}x_{2i} & \sum x_{1i}^2 \end{bmatrix}$$

分母 = $\sum x_{1i}^2 \sum x_{2i}^2 (1 - r_{12}^2)$，当 $r_{12} \to 1$ 时分母 $\to 0$ → $(\boldsymbol{X}'\boldsymbol{X})^{-1} \to \infty$ → 方差 $\to \infty$。

### 口诀

> **不共线 → 分得清 → 方差小**
> **共线性 → 分不清 → 方差大**
> **完全共线 → 完全分不清 → 方差无穷大**

## 三、五大影响（简答必考）⭐⭐⭐

| 序号 | 影响 | 说明 |
|:---:|:-----|:-----|
| ① | **参数方差增大（OLS非有效）** | $Var = \frac{\sigma^2}{\sum x_j^2 (1-r^2)}$，$r \to 1$ 时方差 $\to \infty$ |
| ② | **置信区间变宽** | 标准误大 → 区间宽 → 估计不精确 |
| ③ | **t 检验失效** | $SE \uparrow$ → $t \downarrow$ → 容易接受 $H_0$ |
| ④ | **$R^2$ 高但 t 检验不显著** | 🚨 **多重共线性的经典信号！** F 显著但逐个 t 不显著 |
| ⑤ | **参数对样本极端敏感** | 增减少量样本，系数大幅波动 |

### 与 OLS 三大性质的关系

| 性质 | 受影响？ | 说明 |
|:----|:-------:|:-----|
| **无偏性** | ❌ 不受影响 | $E(\hat{\beta}) = \beta$ 始终成立 |
| **有效性** | ✅ **影响！** | 方差增大，不再是最小方差 |
| **一致性** | ❌ 不受影响 | $n \to \infty$ 时仍趋近真实值 |

## 四、方差膨胀因子（VIF）⭐⭐

$$VIF_j = \frac{1}{1 - R_j^2}$$

其中 $R_j^2$ 是将 $X_j$ 对其他所有解释变量做辅助回归得到的可决系数。

**含义**：
$$Var(\hat{\beta}_j)_{\text{存在共线}} = VIF_j \times Var(\hat{\beta}_j)_{\text{无共线}}$$

**判断标准**：

| VIF 值 | 结论 |
|:------|:-----|
| 1 | 完全不相关 ✅ |
| > 5 | 中等多重共线性 ⚠️ |
| > 10 | 严重多重共线性 ❌ |

**取值范围**：$VIF \in [1, +\infty)$
- $R_j^2 = 0$ → $VIF=1$（最小）
- $R_j^2 \to 1$ → $VIF \to \infty$（完全共线）

## 五、检验方法（简答考点）⭐⭐

| 方法            | 做法                    | 判断标准                       |     |           |
| :------------ | :-------------------- | :------------------------- | --- | --------- |
| **① 相关系数矩阵法** | 计算变量两两相关系数            | $                          | r   | > 0.8$ 可疑 |
| **② 辅助回归法**   | 每个 $X_j$ 对其他变量回归      | $R_j^2$ 接近 1 或 F 显著        |     |           |
| **③ VIF 法**   | $VIF_j = 1/(1-R_j^2)$ | $VIF > 10$ 严重              |     |           |
| **④ 不显著系数法**  | 观察回归结果表               | $R^2$ 高、F 显著、t 多不显著 → 经典信号 |     |           |
| **⑤ 判定系数增量法** | 逐步引入变量看 $R^2$ 增量      | 增量小 → 新变量被已有变量解释           |     |           |

## 六、修正方法（简答考点）⭐⭐

| 方法            | 说明                                                                  |
| :------------ | :------------------------------------------------------------------ |
| **① 剔除变量法**   | 删除引起共线性的次要变量（最常用）                                                   |
| **② 增加样本容量**  | 更多数据"稀释"共线性                                                         |
| **③ 利用先验信息**  | 已知 $\beta_2 = 0.6\beta_3$，代入减少参数                                    |
| **④ 变换模型形式**  | 如差分：$\Delta Y = \beta_2 \Delta X_2 + \beta_3 \Delta X_3 + \Delta u$ |
| **⑤ 逐步回归法** ⭐ | 见下方详细步骤                                                             |
| **⑥ 岭回归**     | 有偏估计（牺牲无偏性换方差降低）                                                    |

### 逐步回归法步骤

1. Y 分别对每个 $X_j$ 做一元回归 → 选 $R^2$ 最大者为基础
2. 逐一引入其他变量
3. **保留条件**：新变量 $R^2$ 提高 **且** t 显著
4. **剔除条件**：新变量导致已有变量 t 不显著
5. 重复到所有变量都显著

## 七、完全 vs. 不完全多重共线性

| 对比项 | 完全 | 不完全 |
|:------|:----|:------|
| 关系 | 精确线性 | 近似线性 |
| $X'X$ 可逆？ | ❌ 不可逆 | ✅ 可逆但接近奇异 |
| OLS 能估计？ | ❌ 无法 | ✅ 能但方差大 |
| 实际常见？ | 少见 | 非常常见 |

| 问题 | 违背假定 | 无偏性 | 有效性 | 核心检验 | 核心修正 |
|:----|:--------|:-----:|:-----:|:--------|:--------|

## 九、习题逐题解析

### 单选题

**1.** 多重共线性时OLS估计量将不具备（ ）→ **C. 有效性**
（无偏性和一致性不受影响，但方差增大 → 不再有效）

**2.** VIF（ ）表示多重共线性严重 → **C. 大于5**
（$VIF > 5$ 中等，$VIF > 10$ 严重）

**3.** 引入无关变量，参数OLS估计量（ ）→ **D. 非有效**
（不影响无偏性，但方差增大）

**4.** $r_{12}=0.5$ 时 $Var(\hat{\beta}_1)$ 是原来的（ ）→ **B. 1.33倍**
（$\frac{1}{1-0.5^2} = \frac{1}{0.75} \approx 1.33$）

**5.** 引入无关解释变量（ ）→ **C. 精度下降**
（无偏性不变，方差增大导致精度下降）

**6.** VIF=10 是什么问题严重 → **C. 多重共线性问题**

**7.** 某个解释变量对其他变量的判定系数接近1 → **A. 多重共线性**

**8.** $X_{1i}=kX_{2i}$ → 完全多重共线性 → **B. 多重共线性**

**9.** 遗漏变量且线性相关 → 有偏且不一致 → **D**

### 多选题

**1.** 哪些可能产生多重共线性？→ **ACE**
（A 资本+劳动 ✅ C 本期+前期收入 ✅ E 施肥量+平方项 ✅）

**2.** 高度多重共线性时（ ）→ **ACD**
（A 难鉴别 ✅ C 精度下降 ✅ D 对样本敏感 ✅）

**3.** 检验多重共线性的统计量 → **ACD**
（A 相关系数 ✅ C VIF ✅ D 特征值 ✅）

### 判断题

**1.** 完全多重共线性时OLS仍是最优线性无偏估计量 → **❌**
（$(X'X)^{-1}$ 不存在，无法估计）

**2.** 高度多重共线时无法评价个别系数显著性 → **✅**

**4.** 变量两两高度相关并不表示高度多重共线性 → **✅**
（多重共线是多个变量的线性关系，两两相关仅是一个信号）

**5.** 如果目的只是预测，多重共线性无害 → **✅**
（$R^2$ 仍然高，预测仍然准确）

**7.** 变量不存在两两高度相关就不存在高度多重共线性 → **❌**
（三个以上变量的线性关系，两两可能不显著）

## 🎯 一句话总结

> **多重共线性 = 解释变量"你中有我" → 分不清各自贡献 → 参数方差膨胀 → OLS不再有效但依然无偏 → 用VIF/辅助回归检验 → 逐步回归/删变量/岭回归修正**
