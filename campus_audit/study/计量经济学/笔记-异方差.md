# 第五章 异方差性笔记

## 一、什么是异方差？

### 同方差（古典假定）

$$Var(u_i) = \sigma^2 \quad (\text{常数})$$

所有观测点的误差项方差相同。

### 异方差（违背假定）

$$Var(u_i) = \sigma_i^2 \neq \text{常数}$$

不同观测点的误差项方差不相等，通常随 $X$ 变化而变化。

### 最直观的例子：储蓄 vs 收入

- **低收入家庭**：收入基本都花掉了，储蓄波动小（方差小）
- **高收入家庭**：可选择储蓄、投资、消费，行为差异大（方差大）

```
储
蓄
↑
|  ·     收入越高，波动越大
| ··     方差变大
|··
|·    ← 收入低时波动小
└─────────→ 收入
```

---

## 二、异方差的类型

| 类型         | 图示            | 说明               |
| :--------- | :------------ | :--------------- |
| **递增异方差**  | 散点图呈喇叭状（左窄右宽） | $X$ 增大时方差增大（最常见） |
| **递减异方差**  | 散点图左宽右窄       | $X$ 增大时方差减小      |
| **复杂型异方差** | 无规律           | 方差随 $X$ 波动变化     |

---

## 三、产生异方差的原因（简答考点）

| 原因                   | 例子                             |
| :------------------- | :----------------------------- |
| **① 模型设定偏误（遗漏重要变量）** | 遗漏的变量被并入误差项，导致误差项方差随该变量变化      |
| **② 截面数据本身特性**       | 截面数据中不同个体规模差异大（大公司vs小公司），波动性不同 |
| **③ 异常值**            | 数据中的极端值拉大方差                    |
| **④ 数据收集技术变化**       | 随时间推移，测量精度改进，误差项方差变小           |
| **⑤ 经济结构变化**         | 改革前后（如1994年分税制），经济行为模式改变       |

---

## 四、异方差对模型的影响（简答必考）⭐⭐⭐

### 与 OLS 三大性质的关系

| 性质 | 受影响？ | 对比多重共线性 |
|:----|:-------:|:-------------|
| **无偏性** | ✅ **不影响** | 一致（都不影响） |
| **有效性** | ❌ **影响！方差不是最小** | 一致（都影响） |
| **一致性** | ✅ **不影响（大样本下仍一致）** | 一致 |

### 具体四大影响

|  序号   | 影响              | 说明                                                                   |
| :---: | :-------------- | :------------------------------------------------------------------- |
| **①** | **OLS 非有效**     | $Var(\hat{\beta})$ 不是最小方差，存在更好的估计                                    |
| **②** | **方差估计有偏**      | 通常的 $\hat{\sigma}^2(\boldsymbol{X}'\boldsymbol{X})^{-1}$ 失效，标准误估计不正确 |
| **③** | **t 检验、F 检验失效** | 标准误错了 → t 值错了 → 检验结论不可信                                              |
| **④** | **预测区间变宽、预测失效** | 参数精度下降 → 预测误差增大                                                      |

## 五、异方差的检验方法（四种，重点）⭐⭐⭐

### 1️⃣ 图示法（最简单，直观判断）

做 **$X$ - $e^2$（残差平方）** 散点图：
- 散点有规律变化（如喇叭形）→ 可能存在异方差
- 散点随机分布 → 同方差

### 2️⃣ Goldfeld-Quandt（GQ）检验 ⭐

| 项目        | 内容                                                                                                                         |
| :-------- | :------------------------------------------------------------------------------------------------------------------------- |
| **适用条件**  | 异方差随某解释变量 $X$ **单调变化**（递增或递减）❌ 不能检验复杂型异方差                                                                                  |
| **基本思想**  | 将样本按 $X$ 排序，去掉中间 $c=n/4$ 个观测，分成前后两组分别回归，比较两组残差平方和                                                                          |
| **检验统计量** | $$F = \frac{RSS_2 / [(n-c)/2 - k]}{RSS_1 / [(n-c)/2 - k]} = \frac{RSS_2}{RSS_1} \sim F(\frac{n-c}{2}-k, \frac{n-c}{2}-k)$$ |
| **判断标准**  | $F > F_{\alpha}$ → 存在异方差、RSS₂ 与 RSS₁ 差异大 → 存在异方差                                                                           |

```
样本排序后：
前 3n/8 个观测 → 回归 → RSS₁（小值组）
中间 n/4 个观测 → 去掉！
后 3n/8 个观测 → 回归 → RSS₂（大值组）

如果递增异方差：RSS₂ >> RSS₁ → F 很大 → 拒绝 H₀
```

### 3️⃣ 怀特（White）检验 ⭐

| 项目        | 内容                                          |
| :-------- | :------------------------------------------ |
| **适用条件**  | **任何形式的异方差**（最通用）                           |
| **辅助回归**  | $e_i^2$ 对所有解释变量、平方项、交叉项做回归，得辅助回归$R^2$       |
| **检验统计量** | $$LM = n \times R^2 \sim \chi^2(k-1)$$      |
| **判断标准**  | $nR^2 > \chi^2_{\alpha}(k-1)$ → 拒绝原假设，存在异方差 |


**辅助回归举例**（二元模型）：
$$e_i^2 = \alpha_0 + \alpha_1 X_{2i} + \alpha_2 X_{3i} + \alpha_3 X_{2i}^2 + \alpha_4 X_{3i}^2 + \alpha_5 X_{2i}X_{3i} + v_i$$

### 4️⃣ ARCH 检验（自回归条件异方差）

| 项目        | 内容                                                          |
| :-------- | :---------------------------------------------------------- |
| **适用条件**  | **时间序列数据**，方差波动的集聚性                                         |
| **辅助回归**  | $e_t^2$ 对 $e_{t-1}^2, e_{t-2}^2, ..., e_{t-p}^2$ 做回归，p为滞后阶数 |
| **检验统计量** | $$LM = (n-p) \times R^2 \sim \chi^2(p)$$                    |
| **判断标准**  | $(n-p)R^2 > \chi^2_{\alpha}(p)$ → 存在 ARCH 异方差               |
$$e_t^2 = \alpha_0 + \alpha_1 e_{t-1}^2 + \alpha_2 e_{t-2}^2 + \cdots + \alpha_p e_{t-p}^2 + v_t$$
### 5️⃣ 戈里瑟（Glejser）检验

| 项目         | 内容             |     |                      |     |                      |     |                          |
| :--------- | :------------- | --- | -------------------- | --- | -------------------- | --- | ------------------------ |
| **基本思想**   | $              | e_i | $ 对 $X$ 的某种函数形式做回归   |     |                      |     |                          |
| **常用辅助函数** | $              | e_i | = \beta X_i + v_i$、$ | e_i | = \beta/X_i + v_i$、$ | e_i | = \beta\sqrt{X_i} + v_i$ |
| **判断**     | 辅助回归显著 → 存在异方差 |     |                      |     |                      |     |                          |

---

## 🌟 四种异方差检验终极对比表

| 对比项            | **GQ 检验**                 | **White 检验**                                                                                                                   | **ARCH 检验**                                                                                        |
| :------------- | :------------------------ | :----------------------------------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------- |
| **辅助回归的被解释变量** | 无（分两组回归，比 $RSS$）          | $e_i^2$                                                                                                                        | $e_t^2$                                                                                            |
| **辅助回归的解释变量**  | 无                         | $X$、$X^2$、交叉项                                                                                                                  | **$e_{t-1}^2, e_{t-2}^2, ..., e_{t-p}^2$**（残差平方的滞后项）                                               |
| **辅助回归方程**     | 不适用                       | $$e_i^2 = \alpha_0 + \alpha_1 X_{1i} + \alpha_2 X_{2i} + \alpha_3 X_{1i}^2 + \alpha_4 X_{2i}^2 + \alpha_5 X_{1i}X_{2i} + v_i$$ | $$e_t^2 = \alpha_0 + \alpha_1 e_{t-1}^2 + \alpha_2 e_{t-2}^2 + \cdots + \alpha_p e_{t-p}^2 + v_t$$ |
| **检验统计量**      | $F = \frac{RSS_2}{RSS_1}$ | $nR^2 \sim \chi^2(k-1)$                                                                                                        | $(n-p)R^2 \sim \chi^2(p)$                                                                          |
| **适用数据**       | 截面数据                      | 截面数据                                                                                                                           | **时间序列数据**                                                                                         |
| **异方差形式**      | **单调**递增/递减               | **任意形式**（最通用）                                                                                                                  | 波动集聚性                                                                                              |
| **优点**         | 最简单直观                     | 最全面，不需事先假设                                                                                                                     | 专门处理时间序列的波动聚集                                                                                      |
| **缺点**         | 只检单调，需选 $c$               | 大样本才可靠，自由度损失大                                                                                                                  | 只适用于时间序列                                                                                           |

> **一句话区分四种检验**：
> - **GQ**：砍两半，比残差
> - **White**：$e^2$ 对 $X$ 全家（$X$、$X^2$、交叉项）回归
> - **ARCH**：$e_t^2$ 对**过去的 $e^2$** 回归（看波动是否聚集）
> - **Glejser**：$|e|$ 试多种函数形式

## 六、White 检验 vs ARCH 检验的区别（简答必考）

| 对比项           | White 检验                  | ARCH 检验                                  |
| :------------ | :------------------------ | :--------------------------------------- |
| **适用数据类型**    | **截面数据**为主                | **时间序列数据**                               |
| **辅助回归的解释变量** | 所有解释变量 $X$、$X^2$、$X$ 的交叉项 | 残差平方的**滞后项** $e_{t-1}^2, e_{t-2}^2, ...$ |
| **检验的异方差类型**  | 一般的异方差                    | 自回归条件异方差（波动的集聚性）                         |
| **统计量**       | $nR^2 \sim \chi^2$        | $(n-p)R^2 \sim \chi^2$                   |

> 一句话区分：**White 看 X 能否解释 e²，ARCH 看过去的 e² 能否解释现在的 e²**

## 七、异方差的修正方法（简答/计算）⭐⭐

### 方法 ①：模型变换法（重点）

**思路**：已知异方差形式 $\sigma_i^2 = \sigma^2 f(X_i)$，两边同除以 $\sqrt{f(X_i)}$

**举例**：若 $Var(u_i) = \sigma^2 X_i^2$

原模型：$Y_i = \beta_0 + \beta_1 X_i + u_i$

两边同除以 $X_i$：
$$\frac{Y_i}{X_i} = \frac{\beta_0}{X_i} + \beta_1 + \frac{u_i}{X_i}$$

此时新误差项的方差：
$$Var\left(\frac{u_i}{X_i}\right) = \frac{Var(u_i)}{X_i^2} = \frac{\sigma^2 X_i^2}{X_i^2} = \sigma^2 \quad \text{✅ 同方差！}$$

### 方法 ②：加权最小二乘法（WLS）

**思路**：对方差小的观测赋予**大权重**，对方差大的观测赋予**小权重**

| 观测 | 方差小 | 数据质量高 | 给大权重 | 更重视 |
|:---:|:------:|:----------:|:-------:|:------:|
| 观测 | 方差大 | 数据波动大 | 给小权重 | 少重视 |

**可选权重**：
$$w_{1i} = \frac{1}{X_i}, \quad w_{2i} = \frac{1}{X_i^2}, \quad w_{3i} = \frac{1}{\sqrt{X_i}}$$

### 方法 ③：对数变换法

**思路**：取对数缩小变量尺度差异

$$Y_i = \beta_0 + \beta_1 X_i + u_i \quad \rightarrow \quad \ln Y_i = \beta_0 + \beta_1 \ln X_i + u_i'$$

> ⚠️ 注意：变换后 $\beta_1$ 的含义变了！原来是边际效应，现在变成了**弹性**。

## 八、习题逐题解析

### 单选题

**1.** 下列哪种方法不是检验异方差的方法？→ **D. 方差膨胀因子检验**
（VIF 是检验多重共线性的，不是异方差）

**2.** 存在异方差时估计模型参数的适当方法？→ **A. 加权最小二乘法**

**3.** WLS 的权数原理：→ **B. 重视小误差，轻视大误差**
（方差小的观测 → 误差小 → 给大权重）

**4.** Glejser 检验得 $|e_i| = 0.28715 x_i + v_i$，WLS 权数应为？→ **C. $1/x_i$**
（因为 $|e_i| \propto x_i$ → $\sigma_i \propto x_i$ → $\sigma_i^2 \propto x_i^2$ → 同除以 $x_i$）

**5.** GQ 检验显著 → **A. 异方差问题**

**6.** 容易产生异方差的数据？→ **C. 横截面数据**
（截面数据个体差异大，如大公司和小公司）

**7.** 存在异方差时应采用？→ **B. 加权最小二乘法**

**8.** $Var(u_i) = \sigma^2 x_i^2$ 时模型变换为？→ **C. $\frac{y_i}{x_i} = \frac{\alpha}{x_i} + \beta + \frac{u_i}{x_i}$**

**9.** $y_i = \beta x_i + u_i$, $Var(u_i) = \sigma^2 x_i^2$ 时 OLS 估计量？→ **B. 无偏但非有效**
（异方差不影响无偏性但影响有效性）

### 多选题

**1.** 哪些很可能存在异方差？
- A. 截面数据建立消费对收入的回归 ✅
- B. 截面数据建立产出对劳动和资本的回归 ✅
- 答案：**AB**

**2.** 异方差下 OLS 的性质？
- A 线性 ✅
- B 无偏性 ✅
- 答案：**AB**（最小方差性❌、有效性❌）

**3.** 异方差性将导致？→ **BCDE**
- B 非有效 ✅
- C 方差估计有偏 ✅
- D 假设检验失效 ✅
- E 预测区间变宽 ✅

**4.** 可以检验异方差的方法？→ **BCDE**
- B GQ检验 ✅ C White检验 ✅ D Glejser检验 ✅ E 帕克检验 ✅
- A DW检验 ❌（那是检验自相关的）

**5.** WLS 估计量具备？→ **BCD**
- B 无偏性 ✅ C 有效性 ✅ D 一致性 ✅

### 判断说明题

**1.** 异方差时 OLS 有偏且不具有最小方差 → **❌**（无偏性不受影响，但有效性受影响）

**2.** 异方差时 t 检验和 F 检验失效 → **✅**

**3.** 异方差时 OLS 估计一定高估了标准差 → **❌**（可能高估也可能低估，取决于异方差形式）

**4.** 残差表现出系统性则说明有异方差 → **✅**
（但也要注意：也可能是模型设定偏误或自相关）

**5.** 遗漏重要变量则残差必定表现出明显趋势 → **✅**

**6.** 异方差时预测失效 → **✅**

## 九、多章对比：异方差 vs 多重共线性 vs 自相关

| 对比项 | 多重共线性 | 异方差 | 自相关 |
|:------|:----------|:------|:-------|
| **违背的假定** | 无多重共线性 | 同方差 | 无自相关 |
| **本质** | 变量间相关 | 方差不恒定 | 误差项相关 |
| **常见数据** | 任何数据 | **截面数据** | **时间序列** |
| **无偏性** | ✅ 不影响 | ✅ 不影响 | ✅ 不影响 |
| **有效性** | ❌ 方差膨胀 | ❌ 非有效 | ❌ 非有效 |
| **检验方法** | VIF / 辅助回归 | **GQ / White / ARCH / Glejser** | **DW / LM** |
| **修正方法** | 删变量 / 逐步回归 | **WLS / 模型变换 / 对数变换** | **广义差分法** |

## 🎯 一句话总结

> **异方差 = 误差项的方差不恒定（随 X 变化）→ OLS 仍无偏但不再有效、t/F 检验失效 → 用 GQ / White / ARCH 检验 → 用 WLS / 模型变换 / 对数变换修正**
