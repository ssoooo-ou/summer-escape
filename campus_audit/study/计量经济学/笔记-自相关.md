# 第六章 自相关性笔记

## 一、什么是自相关？

### 无自相关（古典假定）

$$Cov(u_i, u_j) = 0 \quad (i \neq j)$$

不同观测点的随机误差项之间**不相关**。

### 自相关（违背假定）

$$Cov(u_i, u_j) \neq 0 \quad (i \neq j)$$

不同观测点的随机误差项之间存在**相关关系**。

### 一阶自回归形式 AR(1)

$$u_t = \rho u_{t-1} + v_t$$

其中：
- $\rho$ = 一阶自相关系数，$|\rho| < 1$
- $v_t$ = 满足所有古典假定的新误差项（白噪声）

### $\rho$ = ? 自相关的类型

| $\rho$ 的值 | 自相关类型 | 含义 |
|:----------:|:---------|:------|
| $\rho > 0$ | **正自相关** | 本期误差和上期误差同方向（最常见）|
| $\rho < 0$ | **负自相关** | 本期误差和上期误差反方向 |
| $\rho = 0$ | 无自相关 | 误差独立 ✅ |

---

## 二、用直觉理解自相关

### 最经典的例子：消费函数

$$C_t = \beta_0 + \beta_1 Y_t + u_t$$

$C_t$ = 本期消费，$Y_t$ = 本期收入。

**自相关怎么产生的？**

如果今年收入突然增高，消费也增加（$u_t > 0$）。
但消费有**惯性**——今年消费高了，明年消费也不会立刻降下来（$u_{t+1} > 0$）。

> **今天的 $u_t$ 和明天的 $u_{t+1}$ 是同向的 → 正自相关**

### 画面感：残差图

```
eᵗ
↑
│   ∧      ∧
│  ∧  ∧  ∧  ∧  ∧     ← 正自相关：残差连续几期同号，锯齿但同趋势
│ ∧    ∧      ∧
│
│          ∨      ∨
│         ∨  ∨  ∨  ∨  ∨   ← 负自相关：残差正正负负交替
│        ∨
└────────────────────────→ t
```

---

## 三、产生自相关的原因（简答考点）

| 原因 | 说明 | 例子 |
|:----|:-----|:-----|
| **① 经济惯性** | 经济变量有时间上的持续性 | 消费、GDP、投资——今年的高→明年的也高 |
| **② 模型设定偏误（遗漏变量）** | 遗漏的重要变量有自相关，并入误差项 | 遗漏了消费习惯，它本身有惯性 |
| **③ 数据处理** | 数据插值、平滑等处理引入相关性 | 季度数据由月度数据加总而来 |

## 四、自相关对模型的影响（简答必考）⭐⭐⭐

### 与 OLS 三大性质的关系

| 性质 | 受影响？ | 与异方差/多重共线性对比 |
|:----|:-------:|:---------------------|
| **无偏性** | ✅ **不影响** | 三者都不影响 ✅ |
| **有效性** | ❌ **影响！方差不是最小** | 三者都影响 ❌ |
| **一致性** | ✅ 不影响 | 三者都不影响 ✅ |

### 具体影响

| 序号 | 影响 | 说明 |
|:---:|:-----|:------|
| **①** | **OLS 非有效** | 方差不是最小，存在更好的估计 |
| **②** | **方差估计有偏（通常低估）** | ⚠️ 和异方差不同！自相关通常**低估**标准误，导致 t 值**被夸大** |
| **③** | **t 检验、F 检验失效** | t 值偏大 → 容易错误拒绝 $H_0$（容易误以为显著）|
| **④** | **预测精度下降** | 预测区间不可靠 |

> ⚠️ **对比记忆**：异方差时标准误可高可低；**自相关通常低估标准误** → t 值偏大 → **更容易犯第一类错误（假阳性）**

## 五、DW 检验（计算题必考）⭐⭐⭐

这是自相关检验中**最重要的考点**，必须会查表和判断。

### DW 统计量公式

$$DW = \frac{\sum_{t=2}^{n} (e_t - e_{t-1})^2}{\sum_{t=1}^{n} e_t^2}$$

### DW 与 $\rho$ 的关系

$$DW \approx 2(1 - \hat{\rho})$$

| $\hat{\rho}$ | DW | 含义 |
|:-----------:|:--:|:----|
| $\hat{\rho} = 0$ | $DW \approx 2$ | **无自相关** ✅ |
| $\hat{\rho} = 1$ | $DW \approx 0$ | 完全正自相关 |
| $\hat{\rho} = -1$ | $DW \approx 4$ | 完全负自相关 |

**DW 取值范围**：$0 \leq DW \leq 4$

### DW 判定规则（必背！）⭐⭐⭐

给定显著性水平 $\alpha$，查表得到 $d_L$（下限）和 $d_U$（上限）：

```
          正自相关   无结论区    无自相关      无结论区    负自相关
             │          │          │            │          │
             │          │          │            │          │
             ▼          ▼          ▼            ▼          ▼
  ─────┬─────────┬─────────┬─────────────┬─────────┬─────────┬─────
      0         dL        dU           4-dU      4-dL        4
```

| DW 值范围 | 结论 |
|:---------|:-----|
| $0 < DW < d_L$ | **存在正自相关** ❌ |
| $d_L \leq DW \leq d_U$ | **无结论区**（不确定）❓ |
| $d_U < DW < 4-d_U$ | **无自相关** ✅ |
| $4-d_U \leq DW \leq 4-d_L$ | **无结论区**（不确定）❓ |
| $4-d_L < DW < 4$ | **存在负自相关** ❌ |

### 记忆口诀

> **左正右负，中间无；两头模糊，叫无结论区**

- DW 偏小（接近 0）→ 正自相关
- DW 偏大（接近 4）→ 负自相关
- DW 接近 2 → 无自相关
- 临近 $d_L$ 和 $d_U$ 的那两段 → 无结论区

### 例题

$n=20, k=1, \alpha=0.05, d_L=1, d_U=1.41, DW=2.3$

**判断**：

```
0        1      1.41          2.59(=4-1.41)  3(=4-1)      4
├──正相──┼──?──┼─── 无自相关 ───┼────?──────┼──负相──┤
                    ↑
                  2.3 在这里！
```

$d_U = 1.41 < DW = 2.3 < 4-d_U = 2.59$ → **无自相关** ✅

**对应习题**：单选题第 6 题 → **A**

## 六、DW 检验的局限性（简答考点）

| 局限 | 说明 |
|:----|:------|
| **① 只能检验一阶自相关** | 对高阶自相关（AR(2)、AR(3)）无能为力 |
| **② 必须包含截距项** | 不含截距的模型不能使用 |
| **③ 不能含滞后被解释变量** | 模型中有 $Y_{t-1}$ 时 DW 失效 |
| **④ 存在无结论区** | $d_L \sim d_U$ 和 $4-d_U \sim 4-d_L$ 两段无法判断 |
| **⑤ 样本量不能太小** | 小样本下临界值表可能查不到 |

---

## 七、LM 检验（补充）

当 DW 检验不适用时（如模型含 $Y_{t-1}$、高阶自相关），用 LM 检验（拉格朗日乘数检验）：

**辅助回归**：
$$e_t = \beta_0 + \beta_1 X_t + \rho_1 e_{t-1} + \rho_2 e_{t-2} + \cdots + \rho_p e_{t-p} + v_t$$

**检验统计量**：
$$LM = nR^2 \sim \chi^2(p)$$

> 和 White 检验、ARCH 检验的统计量形式一样，都是 $nR^2$ 服从 $\chi^2$ 分布！

## 八、自相关的修正方法（简答考点）⭐⭐

### 广义差分法（核心方法）

**思路**：把原模型做变换，消除自相关。

**已知 $\rho$ 的情况**：

原模型：$Y_t = \beta_0 + \beta_1 X_t + u_t$，$u_t = \rho u_{t-1} + v_t$

上一期：$Y_{t-1} = \beta_0 + \beta_1 X_{t-1} + u_{t-1}$

两边乘以 $\rho$：$\rho Y_{t-1} = \rho\beta_0 + \rho\beta_1 X_{t-1} + \rho u_{t-1}$

原模型减去上式：
$$Y_t - \rho Y_{t-1} = \beta_0(1-\rho) + \beta_1(X_t - \rho X_{t-1}) + (u_t - \rho u_{t-1})$$

令 $Y_t^* = Y_t - \rho Y_{t-1}$，$X_t^* = X_t - \rho X_{t-1}$，$v_t = u_t - \rho u_{t-1}$：

$$Y_t^* = \beta_0^* + \beta_1 X_t^* + v_t$$

此时 $v_t$ 满足所有古典假定 → **可以放心用 OLS** ✅

### 未知 $\rho$ 的情况（Cochrane-Orcutt 迭代法）

1. 先用 OLS 估计原模型，得到残差 $e_t$
2. 算 $\hat{\rho} \approx 1 - DW/2$
3. 用 $\hat{\rho}$ 做广义差分
4. 重复直到 $\hat{\rho}$ 收敛

### 特殊情况：$\rho \approx 1$ 时用一阶差分法

如果 $\rho \approx 1$（DW ≈ 0），广义差分简化为：

$$\Delta Y_t = \beta_1 \Delta X_t + v_t$$

这就是**一阶差分模型**。

**对应习题**：单选题第 9 题 → 一阶差分适用于 $\rho \approx 1$ → **B**

## 九、习题逐题解析

### 单选题

**1.** 存在序列相关 → **D. $Cov(u_t, u_s) \neq 0 (t \neq s)$**

**2.** DW 检验原假设 → **B. $\rho = 0$**
（原假设是无自相关，备择是有自相关）

**3.** DW 统计量可检验 → **A. $u_t = \rho u_{t-1} + v_t$**
（只能检一阶自回归形式）

**4.** DW 取值范围 → **D. $0 \leq DW \leq 4$**

**5.** DW=4 → **D. 完全负一阶自相关**
（$\rho \approx -1$ → $DW \approx 2(1-(-1)) = 4$）

**6.** $n=20, k=1, DW=2.3, d_L=1, d_U=1.41$ → **A. 不存在一阶自相关**
（$d_U=1.41 < DW=2.3 < 4-d_U=2.59$ ✅）

**7.** 存在序列相关时的估计方法 → **C. 广义差分法**

**8.** 广义差分模型是指 → **D. $Y_t - \rho Y_{t-1} = \beta_0(1-\rho) + \beta_1(X_t - \rho X_{t-1}) + (u_t - \rho u_{t-1})$**

**9.** 一阶差分适用于 → **B. $\rho \approx 1$**
（$\rho=1$ 时 $Y_t - Y_{t-1} = \beta_1(X_t - X_{t-1}) + v_t$）

**11.** $n=30, DW=1.4, d_L=1.35, d_U=1.49$ → **B. 不能判断**
（$d_L=1.35 < DW=1.4 < d_U=1.49$ → 无结论区）

**12.** 明显错误的是 → **B. $\rho=-0.8, DW=-0.4$**
（DW 不能为负！更不可能 -0.4）

**17.** DW 接近 2 → $\rho \approx$ **A. 0**
（$DW \approx 2(1-\rho) = 2 \Rightarrow \rho = 0$）

**18.** $\rho \approx -1$ → DW 近似等于 **D. 4**
（$DW \approx 2(1-(-1)) = 4$）

**20.** $d_L < DW < d_U$ → **D. 存在序列相关与否不能断定**

## 十、三章对比总结（简答王牌考点）

| 对比项 | 多重共线性 | 异方差 | 自相关 |
|:------|:----------|:------|:-------|
| **违背假定** | 无多重共线性 | 同方差 | 无自相关 |
| **本质** | 解释变量相关 | 方差不恒定 | 误差项相关 |
| **常见数据** | 任何数据 | **截面数据** | **时间序列** |
| **无偏性** | ✅ 不影响 | ✅ 不影响 | ✅ 不影响 |
| **有效性** | ❌ 方差膨胀 | ❌ 非有效 | ❌ 非有效 |
| **标准误** | 偏大 | 可高可低 | **通常偏小** |
| **t 检验** | 容易误接受 $H_0$ | 可能高可能低 | **容易误拒绝 $H_0$** |
| **典型信号** | $R^2$ 高但 t 不显著 | 残差图呈喇叭形 | **残差连续同号** |
| **检验方法** | VIF / 辅助回归 | GQ / White / ARCH | **DW / LM** |
| **修正方法** | 删变量 / 逐步回归 | WLS / 模型变换 | **广义差分法** |

## 🎯 一句话总结

> **自相关 = 误差项"有记忆"（本期受上期影响）→ 时间序列数据最常见 → OLS 仍无偏但非有效、标准误被低估 → 用 DW 检验 → 广义差分法修正**
