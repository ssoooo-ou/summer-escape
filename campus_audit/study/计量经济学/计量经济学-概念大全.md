# 一、第一章 · 导论概念

## 1.1 计量经济学（Econometrics）

**定义**：运用统计方法对经济关系进行定量分析的学科，是经济学、统计学和数学的结合。

**研究步骤**（简答必背）：
```
模型设定 → 收集数据 → 参数估计 → 模型检验 → 预测或政策分析
```

## 1.2 四大数据类型

| 数据类型 | 定义 | 例子 |
|:--------|:----|:-----|
| **时间序列数据** | 同一对象、不同时间的数据 | 中国 2000~2023 年 GDP |
| **截面数据** | 同一时点、不同对象的数据 | 2023 年各省人均收入 |
| **面板数据** | 时间 + 截面混合数据 | 各省 2000~2023 年 GDP |
| **虚拟变量数据** | 取值为 0/1 的分类数据 | 性别（男=1，女=0）|

## 1.3 变量分类

| 分类方式 | 变量类型 | 定义 |
|:--------|:--------|:-----|
| **按因果** | 解释变量（自变量 $X$） | 作为原因的一方 |
| | 被解释变量（因变量 $Y$） | 作为结果的一方 |
| **按属性** | 内生变量 | 由模型内部决定（在模型中有方程解释） |
| | 外生变量 | 由模型外部给定（不受模型影响） |

## 1.4 三类可转换模型 ⭐

| 模型名称 | 形式 | 变换方式 | 参数含义 |
|:--------|:----|:--------|:---------|
| **双对数模型**（常数弹性模型） | $\ln Y_i = \beta_0 + \beta_1 \ln X_i + u_i$ | 令 $Y^* = \ln Y,\; X^* = \ln X$ | $\beta_1$ = **弹性**（$X$ 变 1%，$Y$ 变 $\beta_1$%）|
| **半对数模型** | $\ln Y_i = \beta_0 + \beta_1 X_i + u_i$ | 令 $Y^* = \ln Y$ | $\beta_1$ = **半弹性/增长率**（$X$ 变 1 单位，$Y$ 变 $\beta_1 \times 100$%）|
| **倒数模型** | $Y_i = \beta_0 + \beta_1 \cdot \dfrac{1}{X_i} + u_i$ | 令 $X^* = 1/X_i$ | 边际效应递减（如菲利普斯曲线）|

### 🌟 弹性（Elasticity）

**定义**：$X$ 变化 1% 引起 $Y$ 变化百分之几。

**公式**：
$$\text{弹性} = \frac{dY/Y}{dX/X} = \frac{d\ln Y}{d\ln X}$$

> 在双对数模型中，$\beta_1$ 直接就是弹性，且为**常数**，所以也叫**常数弹性模型**。

# 二、第二章 · 简单线性回归概念

## 2.1 核心概念

### 总体回归函数（PRF, Population Regression Function）

**定义**：给定 $X$ 值时 $Y$ 的**条件期望**随 $X$ 变化的函数。

$$E(Y|X_i) = \beta_0 + \beta_1 X_i$$

- $\beta_0$：截距项（当 $X=0$ 时 $Y$ 的平均值）
- $\beta_1$：斜率系数（$X$ 每变动 1 单位，$Y$ 平均变动 $\beta_1$ 单位）
- 这是**理论真实关系**，不可观测

### 总体回归模型

$$Y_i = \beta_0 + \beta_1 X_i + u_i$$

- $u_i$ = 随机误差项（$Y_i$ 与其条件期望的偏差）

### 样本回归函数（SRF, Sample Regression Function）

**定义**：用样本数据估计得到的回归函数。

$$\hat{Y}_i = \hat{\beta}_0 + \hat{\beta}_1 X_i$$

- $\hat{\beta}_0, \hat{\beta}_1$ = 总体参数 $\beta_0, \beta_1$ 的**样本估计量**

### 随机误差项（$u_i$）vs 残差（$e_i$）⭐⭐

| 对比项 | 随机误差项 $u_i$ | 残差 $e_i$ |
|:------|:---------------|:----------|
| **定义** | $u_i = Y_i - E(Y\|X_i)$ | $e_i = Y_i - \hat{Y}_i$ |
| **对应** | 总体回归 | 样本回归 |
| **是否可观测** | ❌ 不可观测 | ✅ 可以计算 |
| **关系** | $e_i$ 是 $u_i$ 的**估计** | — |

### 相关分析 vs 回归分析

| 对比项 | 相关分析 | 回归分析 |
|:------|:--------|:---------|
| 变量地位 | 对称（不分因果） | 不对称（有因果方向）|
| 度量内容 | 关联程度 | 因果数量关系 |
| 典型工具 | 相关系数 $r$ | 回归系数 $\hat{\beta}$ |

### 总体参数 vs 样本统计量

| 对比项 | 总体参数（$\beta_0, \beta_1$） | 样本统计量（$\hat{\beta}_0, \hat{\beta}_1$） |
|:------|:---------------------------|:------------------------------------------|
| 性质 | 常数（唯一确定） | 随机变量（随样本变化）|
| 是否已知 | 未知（需要估计） | 已知（可算出具体数值）|
| 分布 | — | 服从正态分布（大样本下）|

## 2.2 五大古典假定 ⭐⭐⭐（简答必背）

| 假定 | 内容 | 数学表达 |
|:----|:----|:---------|
| **假定 1：线性** | $Y$ 与 $X$ 之间存在线性关系 | $Y_i = \beta_0 + \beta_1 X_i + u_i$ |
| **假定 2：$X$ 非随机** | 解释变量 $X$ 是确定性的 | 重复抽样中 $X$ 固定 |
| **假定 3：零均值** | 随机误差项期望为 0 | $E(u_i) = 0$ |
| **假定 4：同方差 & 无自相关** | 方差相同且互不相关 | $Var(u_i) = \sigma^2$，$Cov(u_i,u_j)=0\;(i\neq j)$ |
| **假定 5：正态性** | 误差项服从正态分布 | $u_i \sim N(0, \sigma^2)$ |

## 2.3 OLS（普通最小二乘法）

**定义**：通过最小化残差平方和 $\sum e_i^2$ 来估计回归参数的方法。

**估计量公式**：

$$\hat{\beta}_1 = \frac{\sum (X_i - \bar{X})(Y_i - \bar{Y})}{\sum (X_i - \bar{X})^2} = \frac{\sum x_i y_i}{\sum x_i^2}$$

$$\hat{\beta}_0 = \bar{Y} - \hat{\beta}_1 \bar{X}$$

> 其中 $x_i = X_i - \bar{X}$，$y_i = Y_i - \bar{Y}$（离差形式）

## 2.4 OLS 回归线的 5 个性质

| 序号 | 性质 |
|:---:|:-----|
| ① | 回归线经过样本均值点 $(\bar{X}, \bar{Y})$ |
| ② | $\bar{\hat{Y}} = \bar{Y}$（$Y$ 的预测值的均值等于实际均值）|
| ③ | $\sum e_i = 0$（残差之和为 0）|
| ④ | $\sum e_i X_i = 0$（残差与 $X$ 不相关）|
| ⑤ | $\sum e_i \hat{Y}_i = 0$（残差与预测值不相关）|

## 2.5 高斯—马尔可夫定理（Gauss-Markov Theorem）⭐⭐⭐

**定义**：在线性回归的古典假定下，OLS 估计量是 $\beta$ 的**最佳线性无偏估计量**（**BLUE**）。

| 性质 | 含义 |
|:----|:------|
| **B**est（最佳） | 在所有线性无偏估计量中方差最小（最有效）|
| **L**inear（线性） | $\hat{\beta}$ 是 $Y$ 的线性函数 |
| **U**nbiased（无偏） | $E(\hat{\beta}) = \beta$ |
| **E**stimator（估计量） | 是对总体参数的估计 |

## 2.6 可决系数 $R^2$（拟合优度）

**定义**：衡量回归模型对数据拟合程度的指标。

**公式**：
$$R^2 = \frac{ESS}{TSS} = 1 - \frac{RSS}{TSS}$$

| 符号 | 含义 | 自由度 |
|:----|:----|:------|
| **TSS** = $\sum (Y_i - \bar{Y})^2$ | 总离差平方和（$Y$ 的总波动）| $n-1$ |
| **ESS** = $\sum (\hat{Y}_i - \bar{Y})^2$ | 回归平方和（$X$ 能解释的部分）| $k-1$ |
| **RSS** = $\sum (Y_i - \hat{Y}_i)^2$ | 残差平方和（$X$ 不能解释的部分）| $n-k$ |

> $TSS = ESS + RSS$（方差分解）

**经济含义**：$Y$ 的总变异中，能被 $X$ 的线性变动解释的比例。
**取值范围**：$0 \leq R^2 \leq 1$

## 2.7 $\hat{\sigma}^2$（随机误差项方差的无偏估计）

**定义**：总体误差方差 $\sigma^2$ 的样本估计量。

$$\hat{\sigma}^2 = \frac{\sum e_i^2}{n-2}$$

其中 $n-2$ 是自由度（估计了 2 个参数 $\hat{\beta}_0$ 和 $\hat{\beta}_1$，损失了 2 个自由度）。

> 简答回归用 $n-2$，多元回归用 $n-k$（$k$ 为总参数个数）✅

## 2.8 回归系数方差和标准误

**斜率系数**：
$$Var(\hat{\beta}_1) = \frac{\sigma^2}{\sum (X_i - \bar{X})^2}, \quad SE(\hat{\beta}_1) = \sqrt{Var(\hat{\beta}_1)}$$

**截距系数**：
$$Var(\hat{\beta}_0) = \sigma^2 \left(\frac{1}{n} + \frac{\bar{X}^2}{\sum (X_i - \bar{X})^2}\right), \quad SE(\hat{\beta}_0) = \sqrt{Var(\hat{\beta}_0)}$$

**标准误的作用**：用于 t 检验的公式分母——$t = \dfrac{\hat{\beta}_j}{SE(\hat{\beta}_j)}$

## 2.9 t 检验（单个系数显著性检验）

**定义**：检验单个回归系数 $\beta_j$ 是否显著不为 0 的统计方法。

**步骤**：
1. $H_0: \beta_j = 0$，$H_1: \beta_j \neq 0$
2. 计算 $t = \dfrac{\hat{\beta}_j}{SE(\hat{\beta}_j)} \sim t(n-2)$
3. 若 $|t| > t_{\alpha/2}(n-2)$，拒绝 $H_0$（系数显著）

## 2.10 F 检验（回归方程整体显著性检验）

**定义**：检验所有解释变量是否**联合显著**的统计方法。

**步骤**：
1. $H_0: \beta_1 = 0$（简单回归中与 t 检验等价）
2. $F = \dfrac{ESS/1}{RSS/(n-2)} = \dfrac{MSR}{MSE}$
3. 若 $F > F_\alpha(1, n-2)$，拒绝 $H_0$

## 2.11 回归预测

**点预测**：直接代入 $\hat{Y}_0 = \hat{\beta}_0 + \hat{\beta}_1 X_0$

**均值区间预测**（$E(Y|X_0)$ 的置信区间）：
$$\hat{Y}_0 \pm t_{\alpha/2}(n-2) \cdot \hat{\sigma} \cdot \sqrt{\frac{1}{n} + \frac{(X_0 - \bar{X})^2}{\sum (X_i - \bar{X})^2}}$$

**个体值区间预测**（$Y_0$ 的预测区间）：
$$\hat{Y}_0 \pm t_{\alpha/2}(n-2) \cdot \hat{\sigma} \cdot \sqrt{1 + \frac{1}{n} + \frac{(X_0 - \bar{X})^2}{\sum (X_i - \bar{X})^2}}$$

> 个体值预测区间 > 均值置信区间（因为多了 $u$ 的不确定性）

# 三、第三章 · 多元线性回归概念

## 3.1 多元线性回归模型

**定义**：包含两个及以上解释变量的线性回归模型。

**形式**：
$$Y_i = \beta_1 + \beta_2 X_{2i} + \beta_3 X_{3i} + \cdots + \beta_k X_{ki} + u_i$$

**矩阵形式**：
$$\mathbf{Y} = \mathbf{X}\boldsymbol{\beta} + \mathbf{u}$$

**OLS 矩阵解**：
$$\hat{\boldsymbol{\beta}} = (\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'\mathbf{Y}$$

## 3.2 偏回归系数（Partial Regression Coefficient）⭐⭐

**定义**：在多元回归中，控制其他解释变量不变的条件下，某个解释变量每变动 1 单位，被解释变量的平均变动量。

$$\beta_j = \frac{\partial E(Y|X)}{\partial X_j} \quad (\text{其他 } X \text{ 不变})$$

> **与简单回归 $\beta$ 的区别**：简单回归的系数衡量的是**总影响**，偏回归系数衡量的是**净影响**（剔除其他变量后的独特贡献）。

## 3.3 FWL 定理（Frisch-Waugh-Lovell Theorem）

**定义**：在多元回归中，$\hat{\beta}_j$ 可以通过"两步法"得到：先剔除 $X_j$ 中其他变量的影响，再用剩余"独特部分"解释 $Y$ 中不能被其他变量解释的剩余部分。

**意义**：
- 揭示了偏回归系数的本质——**"控制其他变量"** 的数学含义
- 解释了多重共线性为什么会导致方差膨胀

## 3.4 调整可决系数 $\bar{R}^2$（Adjusted $R^2$）⭐⭐

**定义**：对普通 $R^2$ 进行自由度调整后得到的拟合优度指标。

$$\bar{R}^2 = 1 - \frac{RSS/(n-k)}{TSS/(n-1)} = 1 - (1-R^2)\frac{n-1}{n-k}$$

| 对比 | $R^2$ | $\bar{R}^2$ |
|:----|:-----|:-----------|
| 增加变量 | 只增不减 | 可能升可能降 |
| 取值范围 | $[0,1]$ | $(-\infty, 1]$（可能为负）|
| 是否惩罚多余变量 | ❌ | ✅ |
| 用途 | 粗略看拟合 | 比较不同变量数量的模型 |

> **设立目的**：惩罚多余的解释变量。新增变量只有贡献足够大时 $\bar{R}^2$ 才会上升。

## 3.5 F 统计量与 $R^2$ 的关系

$$F = \frac{R^2/(k-1)}{(1-R^2)/(n-k)}$$

> 两者同向变化：$R^2$ 越大 → F 越大 → 模型越显著

## 3.6 自由度（Degrees of Freedom）⭐⭐ (高频易错)

| 场景 | 自由度公式 | 原因 |
|:----|:---------|:-----|
| $\hat{\sigma}^2$（简单回归） | $n-2$ | 估计了 $\beta_0, \beta_1$ 两个参数 |
| $\hat{\sigma}^2$（多元回归） | $n-k$ | 估计了 $k$ 个参数（含截距）|
| t 统计量 | $n-k$ | 同 $\hat{\sigma}^2$ 的自由度 |
| F 统计量（分子） | $k-1$ | $k$ 个 $\beta$ 减截距 $\beta_1$ |
| F 统计量（分母） | $n-k$ | 同 t 检验 |
| TSS | $n-1$ | 约束 $\sum(Y_i-\bar{Y})=0$ |
| ESS | $k-1$ | 约束 $\sum(\hat{Y}_i-\bar{Y})=0$ |
| RSS | $n-k$ | 约束来自 $k$ 个正规方程 |

> **速记法**：自由度 = 样本量 - 已估参数数量。每估计一个参数就消耗一个自由度。

# 四、第四章 · 多重共线性概念

## 4.1 多重共线性（Multicollinearity）⭐⭐⭐

**定义**：解释变量之间存在**近似的线性关系**，违背了古典假定中的"无多重共线性"。

**完全多重共线性**：
$$\lambda_2 X_{2i} + \lambda_3 X_{3i} + \cdots + \lambda_k X_{ki} = 0$$
（精确线性关系，$\boldsymbol{X}'\boldsymbol{X}$ 不可逆，OLS 无法估计）

**不完全多重共线性**（实际常见）：
$$\lambda_2 X_{2i} + \lambda_3 X_{3i} + \cdots + \lambda_k X_{ki} + \mu_i = 0$$
（近似线性关系，OLS 可估计但方差极大）

### 产生原因

| 原因 | 说明 |
|:----|:------|
| ① 时间序列的共同趋势 | GDP、消费、投资都随经济周期同向变动 |
| ② 截面数据的内在关联 | 大公司规模大 → 资本、劳动、收入自然相关 |
| ③ 滞后变量的引入 | $X_t$ 和 $X_{t-1}$ 高度相关 |
| ④ 变量选择不当 | 把互相解释的变量都放进模型 |

## 4.2 多重共线性的 5 大影响 ⭐⭐⭐

| 序号 | 影响 | 说明 |
|:---:|:----|:------|
| ① | **参数方差增大（非有效）** | $Var(\hat{\beta}_j) = \dfrac{\sigma^2}{\sum x_j^2 (1-r^2)}$，$r \to 1$ 时方差 $\to \infty$ |
| ② | **置信区间变宽** | 标准误大 → 估计不精确 |
| ③ | **t 检验失效** | $SE \uparrow$ → $t \downarrow$ → 容易错误接受 $H_0$（误认为不显著）|
| ④ | **$R^2$ 高但 t 检验多不显著** | 🚨 **经典信号！** F 显著但逐个 t 不显著 |
| ⑤ | **参数对样本极端敏感** | 增减几个样本，系数大幅波动 |

### 与 OLS 三大性质的关系

| 性质 | 是否受影响 |
|:----|:---------:|
| **无偏性** | ✅ **不影响**（$E(\hat{\beta}) = \beta$ 始终成立）|
| **有效性** | ❌ **影响**（方差膨胀，不是最小方差）|
| **一致性** | ✅ **不影响**（$n \to \infty$ 时仍趋近真实值）|

## 4.3 方差膨胀因子（VIF, Variance Inflation Factor）⭐⭐

**定义**：衡量多重共线性严重程度的指标。

**公式**：
$$VIF_j = \frac{1}{1 - R_j^2}$$

其中 $R_j^2$ 是以 $X_j$ 为被解释变量、其余 $X$ 为解释变量做辅助回归的可决系数。

**含义**：
$$Var(\hat{\beta}_j)_{\text{存在共线}} = VIF_j \times Var(\hat{\beta}_j)_{\text{无共线}}$$

**判断标准**：

| VIF | 结论 |
|:---|:-----|
| 1 | 完全不相关 ✅ |
| 1 ~ 5 | 轻度，可接受 |
| 5 ~ 10 | **中等**多重共线性 ⚠️ |
| > 10 | **严重**多重共线性 ❌ |

> $VIF$ 取值范围：$[1, +\infty)$

## 4.4 多重共线性的检验方法

| 方法 | 做法 | 判断标准 |
|:----|:----|:---------|
| **① 相关系数矩阵法** | 计算变量两两相关系数 | $\|r\| > 0.8$ 可疑 |
| **② 辅助回归法** | 每个 $X_j$ 对其他变量回归 | $R_j^2$ 接近 1 或 F 显著 |
| **③ VIF 法** | $VIF_j = 1/(1-R_j^2)$ | $VIF > 10$ 严重 |
| **④ 不显著系数法** | 观察回归结果 | $R^2$ 高、F 显著、t 多不显著 → 经典信号 |
| **⑤ 判定系数增量法** | 逐步引入变量看 $R^2$ 增量 | 增量小 → 新变量被已有变量解释 |

## 4.5 多重共线性的修正方法

| 方法 | 说明 |
|:----|:------|
| **① 剔除变量法** | 删除引起共线性的次要变量（最常用）|
| **② 增加样本容量** | 更多数据"稀释"共线性 |
| **③ 利用先验信息** | 已知变量间关系，代入减少参数 |
| **④ 变换模型形式** | 如差分 $\Delta Y = \beta_2 \Delta X_2 + \beta_3 \Delta X_3 + \Delta u$ |
| **⑤ 逐步回归法** | 逐个引入/剔除变量，保留显著者 |
| **⑥ 岭回归** | 有偏估计（牺牲无偏性换方差降低）|

# 五、第五章 · 异方差概念

## 5.1 异方差性（Heteroskedasticity）⭐⭐⭐

**定义**：随机误差项的方差不恒定，违背古典假定 4（同方差）。

$$Var(u_i) = \sigma_i^2 \neq \text{常数}$$

### 同方差 vs 异方差

| 对比 | 同方差（满足假定） | 异方差（违背假定） |
|:----|:----------------|:----------------|
| 数学 | $Var(u_i) = \sigma^2$（常数）| $Var(u_i) = \sigma_i^2$（随 $i$ 变化）|
| 图示 | 残差散点图均匀分布 | 残差散点呈喇叭形/规律变化 |
| 常见数据 | 时间序列 | **截面数据** |

### 异方差的类型

| 类型 | 图示 | 说明 |
|:----|:----|:------|
| **递增异方差** | 喇叭状（左窄右宽）| $X$ 增大时方差增大（最常见）|
| **递减异方差** | 左宽右窄 | $X$ 增大时方差减小 |
| **复杂型异方差** | 无规律 | 方差随 $X$ 波动变化 |

### 产生原因

| 原因 | 例子 |
|:----|:-----|
| ① 模型设定偏误（遗漏重要变量） | 遗漏变量被并入误差项，导致方差随该变量变化 |
| ② 截面数据本身特性 | 大公司 vs 小公司，规模差异大 → 波动性不同 |
| ③ 异常值 | 极端值拉大方差 |
| ④ 数据收集技术变化 | 随时间推移测量精度改进，方差变小 |
| ⑤ 经济结构变化 | 改革前后经济行为模式改变 |

## 5.2 异方差对模型的影响

| 序号 | 影响 | 说明 |
|:---:|:----|:------|
| ① | **OLS 非有效** | $Var(\hat{\beta})$ 不是最小方差 |
| ② | **方差估计有偏** | $\hat{\sigma}^2(\boldsymbol{X}'\boldsymbol{X})^{-1}$ 失效，标准误错 |
| ③ | **t、F 检验失效** | 标准误错 → t/F 值错 → 结论不可信 |
| ④ | **预测区间变宽** | 预测精度下降 |

### 与 OLS 三大性质的关系

| 性质 | 是否受影响 |
|:----|:---------:|
| **无偏性** | ✅ **不影响** |
| **有效性** | ❌ **影响**（非有效）|
| **一致性** | ✅ **不影响** |

## 5.3 异方差的四种检验方法 ⭐⭐⭐

### ① GQ 检验（Goldfeld-Quandt Test）

**定义**：将样本按 $X$ 排序后分成两组分别回归，比较两组残差平方和是否显著不同，判断是否存在**单调**异方差。

**适用条件**：异方差随某 $X$ **单调变化**（递增或递减）

**步骤**：
1. 将样本按 $X$ 排序
2. 去掉中间 $c = n/4$ 个观测
3. 前 $3n/8$ 个回归得 $RSS_1$，后 $3n/8$ 个回归得 $RSS_2$
4. 计算统计量：$F = \dfrac{RSS_2/(n-c)/2-k}{RSS_1/(n-c)/2-k} = \dfrac{RSS_2}{RSS_1} \sim F$

**判断**：$F > F_\alpha$ → 存在递增异方差 ✅

### ② 怀特检验（White Test）

**定义**：用 $e_i^2$ 对所有解释变量、平方项、交叉项做辅助回归，通过 $nR^2$ 判断是否存在**任意形式**的异方差。

**适用条件**：任何形式的异方差（**最通用**）

**辅助回归**（二元模型为例）：
$$e_i^2 = \alpha_0 + \alpha_1 X_{1i} + \alpha_2 X_{2i} + \alpha_3 X_{1i}^2 + \alpha_4 X_{2i}^2 + \alpha_5 X_{1i}X_{2i} + v_i$$

**检验统计量**：$LM = nR^2 \sim \chi^2(k-1)$（$k$ 为辅助回归中解释变量个数）

**判断**：$nR^2 > \chi^2_{\alpha}(k-1)$ → 存在异方差 ✅

### ③ ARCH 检验（Autoregressive Conditional Heteroskedasticity）

**定义**：用 $e_t^2$ 对**自身滞后项**做辅助回归，检验时间序列中是否存在"波动集聚性"。

**适用条件**：**时间序列数据**，方差波动的集聚性

**辅助回归**：
$$e_t^2 = \alpha_0 + \alpha_1 e_{t-1}^2 + \alpha_2 e_{t-2}^2 + \cdots + \alpha_p e_{t-p}^2 + v_t$$

**检验统计量**：$LM = (n-p)R^2 \sim \chi^2(p)$（$p$ 为滞后阶数）

**判断**：$(n-p)R^2 > \chi^2_{\alpha}(p)$ → 存在 ARCH 异方差 ✅

### ④ 戈里瑟检验（Glejser Test）

**定义**：用残差的绝对值 $|e_i|$ 对 $X$ 的多种函数形式做辅助回归，检验异方差形式。

**常用辅助函数**：
$$|e_i| = \beta X_i + v_i, \quad |e_i| = \frac{\beta}{X_i} + v_i, \quad |e_i| = \beta \sqrt{X_i} + v_i$$

**判断**：辅助回归**系数显著** → 存在异方差 ✅

### 四种检验对比总表 ⭐⭐⭐

| 对比项 | **GQ 检验** | **White 检验** | **ARCH 检验** | **Glejser 检验** |
|:------|:-----------|:--------------|:-------------|:----------------|
| **被解释变量** | 无（分两组比 $RSS$）| $e_i^2$ | $e_t^2$ | $|e_i|$ |
| **解释变量** | 无 | $X, X^2$，交叉项 | $e_{t-1}^2, e_{t-2}^2, \dots$ | $X$ 的多种函数 |
| **统计量** | $F = \dfrac{RSS_2}{RSS_1}$ | $nR^2 \sim \chi^2$ | $(n-p)R^2 \sim \chi^2$ | t 检验（系数显著？）|
| **适用数据** | 截面数据 | 截面数据 | **时间序列数据** | 截面数据 |
| **异方差形式** | **单调**递增/递减 | **任意形式** | 波动集聚性 | 需尝试多种函数 |
| **一句话记忆** | 砍两半，比残差 | $e^2$ 对 $X$ 全家回归 | $e^2$ 对过去 $e^2$ 回归 | $|e|$ 试多种函数 |

## 5.4 异方差的修正方法

### 模型变换法（已知异方差形式）

**思路**：等式两边同除以 $\sqrt{f(X_i)}$，使新误差项方差变为常数。

**例**：$Var(u_i) = \sigma^2 X_i^2$
$$\frac{Y_i}{X_i} = \frac{\beta_0}{X_i} + \beta_1 + \frac{u_i}{X_i}$$
$$Var\left(\frac{u_i}{X_i}\right) = \frac{\sigma^2 X_i^2}{X_i^2} = \sigma^2 \quad \checkmark$$

### 加权最小二乘法（WLS, Weighted Least Squares）

**定义**：对方差小的观测赋予大权重，方差大的观测赋予小权重，再进行 OLS 估计。

**权重**：$w_i = \dfrac{1}{\sigma_i^2}$

> 实际应用中，若 $|e_i| \propto X_i$ → 取权重 $w_i = 1/X_i^2$

### 对数变换法

$$Y_i = \beta_0 + \beta_1 X_i + u \;\rightarrow\; \ln Y_i = \beta_0 + \beta_1 \ln X_i + u'$$

> ⚠️ 变换后 $\beta_1$ 含义变了：从边际效应变为弹性

# 六、第六章 · 自相关概念

## 6.1 自相关（Autocorrelation / Serial Correlation）⭐⭐⭐

**定义**：不同观测点的随机误差项之间存在相关关系，违背古典假定 5（无自相关）。

$$Cov(u_i, u_j) \neq 0 \quad (i \neq j)$$

### 一阶自回归形式 AR(1)

$$u_t = \rho u_{t-1} + v_t$$

| 符号 | 含义 |
|:----|:------|
| $\rho$ | **一阶自相关系数**，$\|\rho\| < 1$ |
| $v_t$ | **白噪声**（满足所有古典假定的新误差项）|

### $\rho$ 与自相关类型

| $\rho$ | 类型 | 含义 |
|:-----:|:----|:------|
| $\rho > 0$ | **正自相关** | $u_t$ 与 $u_{t-1}$ 同方向（最常见）|
| $\rho < 0$ | **负自相关** | $u_t$ 与 $u_{t-1}$ 反方向 |
| $\rho = 0$ | 无自相关 ✅ | 误差独立 |

### 产生原因

| 原因 | 例子 |
|:----|:-----|
| ① **经济惯性** | 消费、GDP、投资——今年的高→明年的也高 |
| ② **模型设定偏误（遗漏变量）** | 遗漏的变量有自相关，并入误差项 |
| ③ **数据处理** | 数据插值、平滑等引入相关性 |

## 6.2 自相关对模型的影响

| 序号 | 影响 | 说明 |
|:---:|:----|:------|
| ① | **OLS 非有效** | 方差不是最小 |
| ② | **方差估计有偏（通常低估）** | 🚨 **和异方差不同！** 自相关通常**低估**标准误 |
| ③ | **t 检验被夸大** | $t = \hat{\beta}/SE$，$SE$ 被低估 → $t$ 偏大 → **容易错误拒绝 $H_0$**（假阳性）|
| ④ | **预测精度下降** | 预测区间不可靠 |

> ⚠️ **对比记忆**：异方差时标准误可高可低；**自相关通常低估标准误** → t 值偏大 → 更容易犯**第一类错误**

### 与 OLS 三大性质的关系

| 性质 | 是否受影响 |
|:----|:---------:|
| **无偏性** | ✅ **不影响** |
| **有效性** | ❌ **影响**（非有效）|
| **一致性** | ✅ **不影响** |

## 6.3 $\rho$（一阶自相关系数）的估计

**直接公式**：
$$\hat{\rho} = \frac{\sum_{t=2}^n e_t e_{t-1}}{\sum_{t=1}^n e_t^2}$$

**近似公式**（利用 DW）：
$$\hat{\rho} \approx 1 - \frac{DW}{2}$$

## 6.4 DW 检验（Durbin-Watson Test）⭐⭐⭐

**定义**：检验一阶自相关是否存在的最常用方法。

**DW 统计量公式**：
$$DW = \frac{\sum_{t=2}^n (e_t - e_{t-1})^2}{\sum_{t=1}^n e_t^2}$$

**DW 与 $\rho$ 的关系**（近似）：
$$DW \approx 2(1 - \hat{\rho})$$

| $\hat{\rho}$ | $DW$ | 含义 |
|:-----------:|:----:|:----|
| $\hat{\rho} = 0$ | $DW \approx 2$ | **无自相关** ✅ |
| $\hat{\rho} = 1$ | $DW \approx 0$ | 完全正自相关 |
| $\hat{\rho} = -1$ | $DW \approx 4$ | 完全负自相关 |

**取值范围**：$0 \leq DW \leq 4$

### DW 判定规则（必背！）⭐⭐⭐

```
          正自相关   无结论区    无自相关      无结论区    负自相关
             │          │          │            │          │
             ▼          ▼          ▼            ▼          ▼
  ─────┬─────────┬─────────┬─────────────┬─────────┬─────────┬─────
      0         dL        dU           4-dU      4-dL        4
```

| DW 范围 | 结论 |
|:-------|:-----|
| $0 < DW < d_L$ | **存在正自相关** ❌ |
| $d_L \leq DW \leq d_U$ | **无结论区** ❓ |
| $d_U < DW < 4-d_U$ | **无自相关** ✅ |
| $4-d_U \leq DW \leq 4-d_L$ | **无结论区** ❓ |
| $4-d_L < DW < 4$ | **存在负自相关** ❌ |

> **记忆口诀**：左正右负，中间无；两头模糊，叫无结论区

### DW 检验的局限性

| 局限性 | 说明 |
|:------|:-----|
| ① **只能检一阶自相关** | 对 $AR(2)$、$AR(3)$ 无效 |
| ② **必须含截距项** | 无截距模型不能用 |
| ③ **不能含 $Y_{t-1}$** | 滞后被解释变量时 DW 失效 |
| ④ **存在无结论区** | $d_L \sim d_U$ 和 $4-d_U \sim 4-d_L$ 无法判断 |
| ⑤ **样本不能太小** | 小样本查不到临界值 |

## 6.5 LM 检验（Lagrange Multiplier Test / Breusch-Godfrey Test）

**定义**：克服 DW 检验局限性的自相关检验方法，可检验高阶自相关，且模型可含 $Y_{t-1}$。

**辅助回归**：
$$e_t = \beta_0 + \beta_1 X_t + \rho_1 e_{t-1} + \rho_2 e_{t-2} + \cdots + \rho_p e_{t-p} + v_t$$

**检验统计量**：$LM = nR^2 \sim \chi^2(p)$

## 6.6 自相关的修正方法

### 广义差分法（Generalized Difference Method）⭐

**思路**：对模型做变换，消除自相关。

**已知 $\rho$**：
$$Y_t - \rho Y_{t-1} = \beta_0(1-\rho) + \beta_1(X_t - \rho X_{t-1}) + v_t$$

**令**：$Y_t^* = Y_t - \rho Y_{t-1}$，$X_t^* = X_t - \rho X_{t-1}$，$\beta_0^* = \beta_0(1-\rho)$
$$Y_t^* = \beta_0^* + \beta_1 X_t^* + v_t \quad \text{(已消除自相关 ✅)}$$

### 未知 $\rho$ 的情况（Cochrane-Orcutt 迭代法）

1. OLS 估计原模型得残差 $e_t$
2. 算 $\hat{\rho} \approx 1 - DW/2$
3. 用 $\hat{\rho}$ 做广义差分
4. 重复直到 $\hat{\rho}$ 收敛

### 特殊情况：$\rho \approx 1$（一阶差分法）

$$\Delta Y_t = \beta_1 \Delta X_t + v_t$$

# 七、第八章 · 随机解释变量概念

## 7.1 随机解释变量（Stochastic Explanatory Variable）

**定义**：解释变量 $X$ 不再是确定性的，而是随机的（现实中通常如此）。

## 7.2 核心问题：$X$ 与 $u$ 相关

**定义**：当解释变量 $X$ 与随机误差项 $u$ 之间存在相关性时，OLS 估计量不再具有良好的统计性质。

$$Cov(X_i, u_i) \neq 0$$

## 7.3 影响

| $X$ 与 $u$ 的关系 | OLS 性质 |
|:-----------------|:---------|
| $Cov(X, u) = 0$ | **无偏且一致** ✅ |
| $Cov(X, u) \neq 0$ | **有偏且不一致** ❌（比异方差/多重共线性/自相关更严重！）|

> ⚠️ 注意区别：多重共线性、异方差、自相关**都不影响无偏性**，但随机解释变量问题**会导致有偏和不一致**！

## 7.4 解决方法：工具变量法（IV, Instrumental Variables）

**定义**：找到一个变量 $Z$（工具变量），满足：
1. **与 $X$ 高度相关**（相关性条件）
2. **与 $u$ 不相关**（外生性条件）

然后用 $Z$ 替代 $X$ 进行估计。

# 八、三大问题对比总表 ⭐⭐⭐（简答王牌）

## 8.1 核心对比

| 对比项 | 多重共线性 | 异方差 | 自相关 |
|:------|:----------|:------|:-------|
| **违背的假定** | 无多重共线性 | 同方差（假定 4）| 无自相关（假定 5）|
| **本质** | 解释变量相关 | 方差不恒定 | 误差项相关 |
| **常见数据** | 任何数据 | **截面数据** | **时间序列** |
| **无偏性** | ✅ 不影响 | ✅ 不影响 | ✅ 不影响 |
| **有效性** | ❌ 方差膨胀 | ❌ 非有效 | ❌ 非有效 |
| **一致性** | ✅ 不影响 | ✅ 不影响 | ✅ 不影响 |

## 8.2 标准误与检验影响对比

| 对比项 | 多重共线性 | 异方差 | 自相关 |
|:------|:----------|:------|:-------|
| **标准误** | **偏大** | 可高可低 | **通常偏小** |
| **t 值** | 偏小 | 可能偏大/偏小 | 偏大 |
| **t 检验倾向** | 容易**接受** $H_0$（漏判显著）| 不确定 | 容易**拒绝** $H_0$（假阳性）|
| **$R^2$** | 高但 t 不显著 | 不受直接影响 | 可能虚高 |
| **典型信号** | $R^2$ 高但 t 多不显著 | 残差图呈喇叭形 | 残差连续同号 |

## 8.3 检验与修正方法对比

| 对比项 | 多重共线性 | 异方差 | 自相关 |
|:------|:----------|:------|:-------|
| **检验方法** | VIF / 辅助回归 / 相关系数矩阵 | **GQ / White / ARCH / Glejser** | **DW / LM** |
| **修正方法** | 删变量 / 逐步回归 / 增样本 | **WLS / 模型变换 / 对数变换** | **广义差分法** |

# 九、全章核心公式速查

## 第二章（简单回归）

| 公式 | 表达式 |
|:----|:-------|
| OLS 斜率 | $\hat{\beta}_1 = \dfrac{\sum x_i y_i}{\sum x_i^2}$ |
| OLS 截距 | $\hat{\beta}_0 = \bar{Y} - \hat{\beta}_1 \bar{X}$ |
| $\hat{\sigma}^2$ | $\hat{\sigma}^2 = \dfrac{\sum e_i^2}{n-2}$ |
| $Var(\hat{\beta}_1)$ | $\dfrac{\sigma^2}{\sum x_i^2}$ |
| $Var(\hat{\beta}_0)$ | $\sigma^2 \left( \dfrac{1}{n} + \dfrac{\bar{X}^2}{\sum x_i^2} \right)$ |
| $R^2$ | $R^2 = \dfrac{ESS}{TSS} = 1 - \dfrac{RSS}{TSS}$ |
| t 统计量 | $t = \dfrac{\hat{\beta}_1}{SE(\hat{\beta}_1)} \sim t(n-2)$ |
| F 统计量（简答） | $F = \dfrac{ESS/1}{RSS/(n-2)} \sim F(1, n-2)$ |

## 第三章（多元回归）

| 公式 | 表达式 |
|:----|:-------|
| OLS 矩阵解 | $\hat{\boldsymbol{\beta}} = (\boldsymbol{X}'\boldsymbol{X})^{-1}\boldsymbol{X}'\boldsymbol{Y}$ |
| 方差矩阵 | $Var(\hat{\boldsymbol{\beta}}) = \sigma^2 (\boldsymbol{X}'\boldsymbol{X})^{-1}$ |
| $\hat{\sigma}^2$（多元） | $\hat{\sigma}^2 = \dfrac{\sum e_i^2}{n-k}$ |
| $\bar{R}^2$ | $\bar{R}^2 = 1 - (1-R^2)\dfrac{n-1}{n-k}$ |
| F 与 $R^2$ 关系 | $F = \dfrac{R^2/(k-1)}{(1-R^2)/(n-k)}$ |
| t 统计量（多元） | $t = \dfrac{\hat{\beta}_j}{SE(\hat{\beta}_j)} \sim t(n-k)$ |

## 第四章（多重共线性）

| 公式 | 表达式 |
|:----|:-------|
| $Var(\hat{\beta}_1)$（二元） | $\dfrac{\sigma^2}{\sum x_{1i}^2 (1-r_{12}^2)}$ |
| VIF | $VIF_j = \dfrac{1}{1-R_j^2}$ |

## 第五章（异方差）

| 公式 | 表达式 |
|:----|:-------|
| GQ 统计量 | $F = \dfrac{RSS_2}{RSS_1} \sim F$ |
| White 统计量 | $LM = nR^2 \sim \chi^2(k-1)$ |
| ARCH 统计量 | $LM = (n-p)R^2 \sim \chi^2(p)$ |
| WLS 权重 | $w_i = \dfrac{1}{\sigma_i^2}$ |

## 第六章（自相关）

| 公式 | 表达式 |
|:----|:-------|
| DW 统计量 | $DW = \dfrac{\sum_{t=2}^n (e_t - e_{t-1})^2}{\sum_{t=1}^n e_t^2}$ |
| DW 与 $\rho$ | $DW \approx 2(1-\hat{\rho})$ |
| 广义差分 | $Y_t^* = Y_t - \rho Y_{t-1}$，$X_t^* = X_t - \rho X_{t-1}$ |
